## 摘要
同策略蒸馏（OPD）具有离线蒸馏和强化学习中所没有的特性：教师监督质量取决于学生能力。不连贯的轨迹会产生噪声梯度；已掌握的标记会产生冗余梯度。这在三个尺度（标记、训练阶段和提示）上造成浪费，但现有方法却采用统一监督。我们提出SEAD，利用熵作为统一探测器，在三个尺度上感知这种依赖能力的退化：（1）联合师生熵将标记划分为接受定制散度或零梯度的区域（约50%被跳过）；（2）余弦调度从正向KL退火到逆向KL，随着能力增长而变化；（3）能力门控的课程从易到难引入提示。这些组件共生地必要：标记选择需要连贯的轨迹（课程），退火需要单调改进（也是课程）。在OLMo-3（7B到32B）上，SEAD在六个数学基准测试中相比普通OPD平均准确率提升+4.8，消融实验证实了超加性交互。

## Abstract
On-policy distillation (OPD) has a property absent in offline distillation and RL: teacher supervision quality depends on student competence. Incoherent rollouts yield noisy gradients; already-mastered tokens yield redundant ones. This creates waste at three scales (tokens, training phases, and prompts) yet existing methods supervise uniformly. We introduce SEAD, which uses entropy as a unified probe of this competence-dependent degradation at three scales: (1) joint teacher-student entropy partitions tokens into zones receiving tailored divergences or zero gradient (approx. 50% skipped); (2) a cosine schedule anneals from forward to reverse KL as competence grows; (3) a competence-gated curriculum introduces prompts easy-to-hard. These components are symbiotically necessary: token selection requires coherent rollouts (curriculum), annealing requires monotonic improvement (also curriculum). On OLMo-3 (7B to 32B), SEAD achieves +4.8 avg accuracy over vanilla OPD across six math benchmarks, with ablations confirming super-additive interactions.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：On-policy distillation（OPD）具有独特性质——教师监督质量依赖于学生当前的能力。具体表现为三个尺度的浪费：
  - **Token 级**：约 50% 的 token 已被师生双方掌握（高确定性），监督计算冗余；剩余 token 中，有些需要模式覆盖（FKL），有些需要模式追求（RKL）。
  - **时间级**：最优散度随学生能力增长从覆盖模式转向追求模式，但现有方法采用离散的二阶段切换。
  - **提示级**：超出学生能力的问题产生不连贯轨迹，教师监督退化为噪声。
- **现有不足**：现有 OPD 方法在所有 token、阶段和提示上使用统一的监督策略（如同一种散度、无课程），忽略了监督质量随能力变化的依赖。
- **论文主张**：通过熵作为统一可观测信号，在三个尺度上自适应地分配计算资源，只在监督质量高的地方进行更新。

## 2. 方法论：核心思想、关键技术细节

### 核心思想：SEAD（Sparse Entropy-Adaptive Divergence）

利用联合师生熵（teacher-student entropy）作为“能力-监督质量”的探针，将问题分解为三个共生组件，协同工作。

### 关键技术细节

#### (a) Token 级：稀疏熵自适应散度（Token-level Selection）

- 根据每个 token 的师生熵 \(H_{\text{te}}(t)\) 和 \(H_{\theta}(t)\) 将 token 分为三个区域（通过百分位阈值，默认 ρA=50%, ρB=40%, ρC=10%）：
  - **Zone A（跳过）**：两者熵均低 → 零梯度，约 50% token 被跳过。
  - **Zone B（RKL）**：教师低熵、学生高熵 → 使用逆向 KL 使学生向教师模式集中。
  - **Zone C（FKL）**：教师高熵（真正推理分支点） → 使用前向 KL 保持多样化。
- 损失：\(L_{\text{token}}^{\text{SEAD}} = \frac{1}{|B\cup C|} \left( \sum_{t\in B} L_{\text{RKL}}(t) + \lambda \sum_{t\in C} L_{\text{FKL}}(t) \right)\)，λ 平衡两种散度。
- 理论保证（Theorem 1）：在低熵假设下，跳过 Zone A 带来的梯度误差有上界 \(O(s G \sqrt{\tau} \log |V|)\)，即近似无损。

#### (b) 时间级：能力驱动的退火（Competence-Driven Annealing）

- 随着训练进行，活动 token 的组成变化：早期 Zone C（两者都不确定）占主导，晚期 Zone B（教师自信）占主导。退火系数 α 从 0.8 线性/余弦衰减到 0.0：
  \[
  \alpha(t_{\text{step}}) = \alpha_{\text{end}} + \frac{\alpha_{\text{start}} - \alpha_{\text{end}}}{2} \left(1 + \cos\left(\frac{t_{\text{step}}}{T_{\text{total}}} \pi\right)\right)
  \]
- α 控制 Zone C 中 FKL 项的权重：早期强调探索（高 α），晚期强调精炼（低 α）。

#### (c) 提示级：能力门控课程（Competence-Gated Curriculum）

- 事先用学生初始模型采样 K 次（K=8），计算每个提示的通过率 \(p_i\)，定义难度 \(d_i = 1 - p_i\)。
- 定义能力函数 \(c(t) = \min(1, c_0 (1 + t/T)^p)\) 单调增长。每个训练步只选择难度 \(d_i \leq c(t)\) 的提示构成 eligible set。
- 理论基础（Theorem 2）：课程保证了所有合格提示上学生能力有下界 \(c>0\)，从而梯度信号不会消失。与随机均匀采样相比，收敛速度提升因子 \(\Omega((c/\bar{\phi}_U)^2)\)，该优势仅在 OPD 中存在（因监督质量随能力变化），RL 和离线 KD 中无此优势。

#### (d) 组件共生关系（Symbiotic Necessity）

- 课程提供连贯轨迹 → 熵景观可靠 → token 级分区正确 → 退火能正确跟踪。三个组件互相依赖，消融实验证明超加性（super-additive）增益。

### 算法流程（Algorithm 1）

1. 预计算每个提示的难度 \(d_i\)（基于初始学生通过率）。
2. 在每个训练步：
   - 根据当前能力 \(c(t)\) 确定 eligible 提示集。
   - 采样 batch，由学生生成 rollout。
   - 教师打分，计算每 token 师生熵。
   - 通过百分位阈值分配 Zone A/B/C。
   - 根据退火系数 α 计算损失。
   - 更新学生参数（使用 clipped gradient）。

## 6. 主要结论与发现

1. **SEAD 显著提升 OPD 性能**：在 OLMo-3 上，从 Vanilla OPD 的 59.2 提升至 64.0 平均准确率（+4.8），在 Nemotron 上也观察到一致趋势。
2. **组件超加性协同**：单独 Token Zones 或 KL Annealing 几乎无提升；两者结合产生 20 倍放大（+4.5）；再加上课程进一步提升至 +4.8。完整组合的增益大于各增益之和。
3. **Token 级稀疏性几乎无损**：跳过 50% token 仅损失约 10% 损失信号，梯度方向近似一致。
4. **课程是 OPD 特有需求**：课程收敛速度比均匀采样快 \(\Omega((c/\bar{\phi}_U)^2)\) 倍，该优势在 RL 和离线 KD 中不存在。
5. **熵崩塌缓解**：SEAD 维持学生熵稳定，而 Vanilla OPD 熵持续下降（0.39→0.31）。
6. **Zone 分析符合直觉**：Zone A 多是句法和确定性计算，Zone B 是推理过渡，Zone C 是真正分支点（策略分支 100% 被分配到 B 或 C 区，从未进入 Zone A）。

## 7. 优点

1. **统一框架**：用熵作为单一可观测信号，同时解决三个尺度的效率问题，设计简洁优雅。
2. **共生设计**：明确三个组件互相预条件，消融实验证实超加性，理论分析（Theorem 1 & 2）为组件必要性提供数学保证。
3. **理论扎实**：提供梯度误差界、收敛加速比等证明，增强了方法的可信度。
4. **实验深入**：不仅有主实验结果，还有详细的 token 级分析（损失集中度、语义分类、熵空间分布）和训练动态，揭示方法为什么有效。
5. **计算高效**：跳过 50% token 几乎不损失训练信号，可显著降低训练成本（约减半 token 计算量）。
6. **可推广性**：在两种不同模型族（OLMo, Nemotron）上验证，表明方法不局限于特定架构。
7. **开放问题贡献**：首次将课程学习应用于 OPD，回应了先前文献（Zhao et al., 2026; Li et al., 2026）指出的开放问题。
