## 摘要
大语言模型在自回归推理中速度缓慢。虽然自推测解码加速了这一过程，但其效率受到固定退出层和推测长度等静态配置的限制。我们将这一优化重新定义为马尔可夫决策过程，并提出了LEDE框架，该框架使用离线强化学习。LEDE学习一个策略，根据生成序列每一步的局部上下文动态选择最优的退出层和推测长度，平衡计算成本和草案质量。在Llama-2和Llama-3模型上的综合评估表明，LEDE相比自回归解码实现了高达2.0倍至2.7倍的加速，并且相比静态推测基线额外提供了17%的加速。

## Abstract
Large Language Models suffer from slow autoregressive inference. While self-speculative decoding accelerates this process, its efficiency is hampered by static configurations like fixed exit layers and speculation lengths. We reframe this optimization as a \textbf{Markov Decision Process} and propose \textbf{LEDE}, a framework that uses offline reinforcement learning. LEDE learns a policy to dynamically select the optimal exit layer and speculation length based on the local context of the generated sequence at each step, balancing computational cost and draft quality. Comprehensive evaluations on Llama-2 and Llama-3 models show LEDE achieves up to a $2.0\times$$\sim$$2.7\times$ speedup over autoregressive decoding and and provides an additional 17\% speedup over the static speculative baselines.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究动机**：大型语言模型（LLM）的自回归推理过程非常缓慢。自推测解码（Self-Speculative Decoding, SSD）通过利用模型早期层作为内部“草稿模型”来加速推理，但现有SSD方法依赖**静态配置**（固定退出层和固定推测长度），无法适应不同生成步骤的上下文难度差异，导致效率低下。
- **核心问题**：如何根据当前生成序列的局部上下文，动态且自适应地选择最优的退出层和推测长度，以平衡计算成本与草稿质量。
- **整体含义**：本文提出一种基于强化学习（RL）的框架LEDE，将动态控制问题建模为马尔可夫决策过程（MDP），通过离线RL学习一个策略，在推理过程中实时决策，从而显著提升LLM推理速度。

### 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：将自推测解码中的退出层选择和推测长度控制统一建模为**马尔可夫决策过程（MDP）**，使用**离线强化学习**（Offline RL）训练一个Deep Q-Network（DQN）策略，根据模型中间层的概率分布特征动态决定“继续”或“退出”，实现自适应推理。
- **关键组件**：
  - **状态空间**：在每个候选层l，提取三个特征构成状态向量\( s_t \)：
    - **Token置信度**：Top-k token的平均负对数概率（k=5）。
    - **Token熵**：下一token预测分布的信息熵。
    - **平均Trace Token置信度**：历史窗口内Top-1概率的平均负对数概率（窗口大小T=20）。
  - **动作空间**：在每个候选层，策略输出二值动作：`continue`（继续下一层）或`exit`（选定当前层为退出层）。
  - **奖励函数**：
    - 如果退出且草稿token被接受：\( R = 1 + (L - l^*)/L \)（正奖励，鼓励浅层退出且正确）。
    - 如果退出但草稿被拒绝：\( R = -2 + (L - l^*)/L \)（负惩罚）。
    - 如果继续：\( R = -0.01 \)（小惩罚，鼓励尽早退出）。
  - **策略优化**：采用**DQN**，使用一个小型MLP作为Q网络，通过n-step Temporal Difference（TD）误差（n=3，折扣因子γ=0.99）离线训练；经验来自初始探索阶段收集到的转移元组，存储在回放缓冲区中。
  - **双阶段控制**：
    - 第一阶段：通过逐层评估，动态选择最佳退出层\( l^* \)。
    - 第二阶段：在\( l^* \)层开始生成草稿，每生成一个草稿后，再次用策略决定是否继续生成草稿还是终止验证，从而自适应控制推测长度。
