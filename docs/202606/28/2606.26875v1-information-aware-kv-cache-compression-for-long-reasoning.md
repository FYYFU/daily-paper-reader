## 摘要
推理能力在大语言模型（LLM）中发展迅速，导致在预填充和解码阶段键值（KV）缓存规模不断增加。现有的KV缓存压缩方法主要依赖注意力权重来估计令牌重要性。虽然注意力能有效捕捉上下文相关性，但它忽略了与预测不确定性和令牌信息量相关的互补信息论信号。在本文中，我们从前瞻性角度重新审视令牌重要性，并引入	extit{前向影响}（Forward Influence）这一度量，评估压缩令牌对未来上下文的影响。我们的分析揭示，由注意力分数选择的令牌主要影响邻近上下文，而与高预测不确定性相关的令牌对遥远未来上下文具有显著更强的影响。基于这一观察，我们提出	extbf{InfoKV}，一种融合信息论信号、感知熵的KV缓存压缩框架。它结合令牌级预测不确定性与层间表示演化，并在推理过程中将得到的熵分数与注意力分数整合。在Llama-3.1、Llama-3.2和DeepSeek-R1上的长上下文推理基准实验中，InfoKV在长预填充和解码场景中均持续优于现有的基于注意力的KV压缩方法。

## Abstract
Reasoning capability has advanced rapidly in large language models (LLMs), leading to an increasing size of key-value (KV) cache in both prefilling and decoding stages. Existing KV cache compression methods mainly rely on attention weights to estimate token importance. While attention effectively captures contextual relevance, it overlooks complementary information-theoretic signals related to predictive uncertainty and token informativeness. In this paper, we revisit token importance from a forward-looking perspective and introduce \textit{Forward Influence}, a metric that measures how compressed tokens affect future contexts. Our analysis reveals that tokens selected by attention scores mainly influence nearby contexts, whereas tokens associated with high predictive uncertainty exhibit substantially stronger influence on distant future contexts. Based on the observation, we propose \textbf{InfoKV}, an entropy-aware KV cache compression framework that incorporates information-theoretic signals. It combines token-level predictive uncertainty with layer-wise representation evolution and integrates the resulting entropy scores with attention scores during reasoning. Experiments on long-context reasoning benchmarks with Llama-3.1, Llama-3.2, and DeepSeek-R1 demonstrate that InfoKV consistently outperforms existing attention-based KV compression methods in both long prefilling and decoding scenarios.

论文聚焦于大语言模型（LLM）在长上下文推理场景中的 KV 缓存膨胀问题。随着推理长度的增加，KV 缓存占用的内存和计算开销呈线性增长，严重制约了部署效率。现有主流的 KV 缓存压缩方法（如 SnapKV、PyramidKV、Expected Attention 等）几乎全部依赖注意力权重（attention weights）来估计 token 的重要性，即认为能被近期上下文高度关注的 token 才是重要的。然而，论文指出这种机制本质上是 **后顾的（backward-looking）**，它只能捕捉局部依赖性，却忽略了那些对**未来推理轨迹**可能至关重要的 token。这些 token 可能当前未被近期上下文关注，但携带着高预测不确定性（高信息量），对遥远的未来生成步骤有持久影响。因此，论文从**前瞻性（forward-looking）**角度出发，引入信息论信号（熵），提出信息感知的 KV 缓存压缩方法 **InfoKV**，旨在同时兼顾局部和长期重要性。

## 二、论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 重新定义 token 重要性：不仅依赖注意力分数，还要结合 token 的**预测不确定性（熵）**和**层间表示演化（layer-wise representation evolution）**。
- 熵高的 token 通常是内容词（名词、动词、形容词），携带丰富语义，且对远方未来上下文影响更大。
- 通过结合注意力（短期局部相关）和熵（长期全局信息量）得到综合重要性分数，指导 KV 缓存压缩。

### 关键技术细节

#### 1. 前向影响（Forward Influence）度量
- 定义：移除某个 token \(x_i\) 后，未来一段上下文 \(x_{l_c:r_c}\) 上 KL 散度的平均变化。
  \[
  I_{l_c:r_c}(x_i) = \frac{1}{r_c - l_c + 1} \sum_{n=l_c}^{r_c} D_{KL}\left( p_\theta(x_n|x_{<n}) \parallel p_\theta(x_n|x_{<n}\setminus\{x_i\}) \right)
  \]
- 用于分析不同评分策略（注意力、熵、组合）对短期和长期的影响差异。

#### 2. 熵的获取
- 对每个 token \(x_n\)，利用模型预测分布计算 **Top-k 受限熵**（只取概率最高的 k 个 token），避免低概率 token 的噪声。
  \[
  H(x_n) = -\sum_{\hat{x}_n \in \text{Top-}k} \hat{p}(x_n) \log \hat{p}(x_n)
  \]

#### 3. 层间表示演化（表示距离）
- 计算同一 token 在**早期层 \(l\)** 和**最终层 \(L\)** 的隐藏表示之间的余弦距离：
  \[
  D_i^{(l)} = 1 - \cos\left(h_i^{(l)}, h_i^{(L)}\right)
  \]
- 距离越大，说明该 token 在深层仍有较大语义变化，携带未收敛的信息。

#### 4. 组合熵分数
- 将表示距离与熵相乘，得到每层的熵分数：
  \[
  E_i^{(l)} = D_i^{(l)} \cdot H_i
  \]
- 加入偏置 \(\tau\) 避免最后层为 0。

#### 5. 最终重要性分数
- 对每层，将注意力分数 \(A_i^{(l)}\) 与熵分数的 softmax 版本加权求和：
  \[
  S_i^{(l)} = \alpha \cdot A_i^{(l)} + (1-\alpha) \cdot \text{Softmax}(\mathbf{E}^{(l)})_i
  \]
- 其中 \(\alpha\) 控制平衡（实验中默认 \(\alpha=0.9\) 或 0.95）。

#### 6. 压缩策略
- 对每层保留重要性分数最高的 token，支持均匀预算或自适应预算（根据各层总熵分配）。

5. **表示距离提供层特异性**：早期层到最终层的表示距离能反映 token 语义收敛程度，进一步丰富了熵分数。
