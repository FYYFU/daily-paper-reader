## 摘要
混合智能体（MoA）架构通过将多个LLM智能体组织成分层推理流水线，改进了推理时规模化。然而，现有的MoA变体无法随着深度增加而维持收益，表现出性能退化、早期平台期或饱和。我们提出ReM-MoA，一种记忆增强的MoA框架，通过两种机制维持规模化：（1）排名推理记忆，它使用比较性评审智能体持久存储并排名所有层的推理轨迹；（2）策展多样化记忆路由方案，它将不同智能体暴露于成功和失败轨迹的不同组合中，在传播高质量推理的同时保留探索多样性。我们进一步引入可选的跨领域评审者蒸馏流水线，通过前沿模型监督提高排名质量。在涵盖数学、形式逻辑、代码、知识和常识的五项推理基准上，ReM-MoA在深度和宽度规模化上均持续优于先前的MoA变体，并且其优势随着深度增加而扩大，确立了结构化跨层推理记忆作为可扩展多智能体推理的关键缺失机制。

## Abstract
Mixture-of-Agents (MoA) architectures improve inference-time scaling by organizing multiple LLM agents into layered reasoning pipelines. However, existing MoA variants fail to sustain gains as depth increases, exhibiting degradation, early plateauing, or saturation. We propose ReM-MoA, a memory-augmented MoA framework that sustains scaling through two mechanisms: (1) a Ranked Reasoning Memory that persistently stores and ranks reasoning traces from all layers using a comparative Reviewer Agent, and (2) a Curated Diversified Memory Routing scheme that exposes different agents to distinct combinations of successful and failed traces, preserving exploration diversity while propagating high-quality reasoning. We further introduce an optional multi-domain Reviewer distillation pipeline that improves ranking quality through frontier-model supervision. Across five reasoning benchmarks spanning math, formal logic, code, knowledge, and commonsense, ReM-MoA consistently outperforms prior MoA variants across both depth and width scaling, and its advantage widens with depth, establishing structured cross-layer reasoning memory as a key missing mechanism for scalable multi-agent inference.

## 1. 核心问题与整体含义（研究动机和背景）

**研究动机**：现有 Mixture-of-Agents（MoA）架构通过将多个 LLM 智能体组织成分层推理流水线来改进推理时规模化。然而，随着深度（层数）增加，现有变体均无法持续获益：标准 MoA 性能退化，RMoA 早期平台期，AttentionMoA 饱和。其根本原因包括：原始拼接传播错误、仅局部通信丢失早期高质量轨迹、智能体之间探索多样性崩溃。

**整体含义**：本文认为，要实现可扩展的多智能体推理，需要同时保留高质量推理轨迹（质量）和智能体之间的探索多样性（多样性），即**结构化跨层推理记忆**是关键缺失机制。

## 2. 方法论：核心思想、关键技术细节、算法流程

**核心思想**：提出 ReM-MoA，包含两种机制——

- **Ranked Reasoning Memory（排名推理记忆）**：每层使用一个比较性 Reviewer Agent 对同一层的 N 个提议者轨迹进行对比评分（1-10 分）并生成本条理由，然后将轨迹-评分-理由三元组追加存储到全局记忆 M 中。记忆在所有后续层可读，且保持有序（按评分排序）。
- **Curated Diversified Memory Routing（策展多样化记忆路由）**：从第 2 层起，每层的每个提议者接收不同的参考集，这三个参考集家族包括：
  - Top n：最高分轨迹（正强化）
  - Bot n：最低分轨迹（失败模式聚焦）
  - Con n：Top a ∪ Bot b（对比学习）
  每个智能体分配不同组合，确保同一层内智能体参考不同内容，维持多样性。
- **可选增强**：通过前沿模型（GPT-5.5）蒸馏训练一个更小的 Reviewer（使用 LoRA 微调 Qwen2.5-7B），多领域训练使 Reviewer 学会泛化的评估能力。

**算法流程**（文字说明）：
1. 第一层：输入问题 x，N 个提议者各自生成轨迹 r_{1,j}。
2. Reviewer Agent 接收所有 N 条轨迹，输出评分 s_{1,j} 和理由 ρ_{1,j}，存入记忆 M_1。
3. 后续第 l 层（l≥2）：
   - 路由模块从记忆 M_{≤l-1} 中为每个提议者选择不同的参考集 S_{l,j}（来自 Top/Bot/Con 家族）。
   - 提议者根据 x 和参考集生成新轨迹 r_{l,j}。
   - Reviewer 再次对这些轨迹评分，存入记忆 M_l。
4. 所有 L 层结束后，聚合器从记忆中选择参考集，综合生成最终答案。
