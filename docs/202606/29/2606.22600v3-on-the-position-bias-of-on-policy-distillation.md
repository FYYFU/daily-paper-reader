## 摘要
同策略蒸馏（OPD）通过教师提供的密集、词元级监督提高了标准强化学习的学习效率。在OPD的标准KL目标中，词元级损失被均匀平均，意味着所有词元具有相同权重。然而，我们发现并非所有词元都是平等的：随着学生轨迹变长，它们与教师分布的偏差增大，导致后期位置的监督质量降低。因此，仅使用前30%词元的OPD可以与使用所有词元的OPD性能相当，而仅使用后30%词元的OPD几乎学不到任何东西。本文通过约束优化的视角对这一问题的原理进行了理解。基于这些见解，我们推导出重要性加权同策略蒸馏（IW-OPD），其中分配给每个词元的权重取决于学生与教师分布之间的累积差异，自然地提高早期词元的权重并降低偏差较大的后期词元的权重。我们表明，IW-OPD的收敛速度明显快于OPD，学习效率更高，并且在同规模和跨规模设置中均比标准OPD取得更好的最终性能，在AIME-2025上性能提升高达6.9分。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

## 1. 核心问题与整体含义（研究动机和背景）

- **标准OPD的均匀加权假设**：同策略蒸馏（OPD）通过教师模型在每一 token 上提供密集监督，损失函数为所有 token 的 KL 散度之和的期望（平均），隐含了所有 token 同等重要。
- **位置偏差的发现**：随着学生生成的序列变长，其分布逐渐偏离教师的分布，导致后部 token 的监督质量急剧下降。实验表明，仅用前 30% token 训练的 OPD 与用全部 token 性能相当，而仅用后 30% token 几乎无法学习（图 1）。
- **研究动机**：这种位置偏差导致 OPD 的梯度预算被低质量的后期监督浪费，学习效率低下。本文旨在理解和量化位置偏差，并提出更高效的加权策略。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 核心思想
- 从**约束优化视角**出发：将 OPD 视为在有限局部更新预算（KL 半径 ρ）内，向教师分布做投影。
- 最优策略应**按教师-学生似然比 α 次幂重采样**，自动提高教师相容前缀的采样权重，相当于给早期 token 更高权重、后期 token 更低权重。

### 关键技术细节
- **重要性权重的推导**：通过拉格朗日乘子法解约束优化问题，得到最优投影策略 q*，其形式为 πθ 乘以归一化的似然比 r(y)^α。该比值的累积对应于前缀的教师-学生分布差异。
- **稳定化估计策略**（避免直接使用原始似然比导致的方差爆炸）：
  1. **对数缩放**：使用 token 级 KL 优势的累积和代替原始乘积。
  2. **符号校正**：取绝对值求和，防止正负抵消（使用 `|A_OPD_k|`）。
  3. **归一化**：对每个样本内的权重线性压缩至 [0,1]。
  4. **与标准 OPD 插值**：最终权重 = 1 + γ × 归一化权重的线性变换（γ=0.5 默认）。
- **算法流程（Algorithm 1）**：
  1. 从当前学生策略采样轨迹；
  2. 计算每个 token 的 OPD 优势 `A_OPD_t = -(log πθ - log πT)`；
  3. 计算前缀累积绝对差异，并应用归一化与插值得到 `r_IW-OPD_t`；
  4. 将 `A_OPD_t` 乘以 `sg[r_IW-OPD_t]`（stop-gradient）得到 IW-OPD 优势；
  5. 使用 PPO 风格的目标更新学生策略（含重要性裁剪）。

- **IW-OPD 一致优于标准 OPD**：在所有教学-学生配对和所有基准上，最终性能（收敛后）获得 0.4~6.9 个点的绝对提升。

- **开源拓展性**：明确使用公开模型和数据，可与未来 OPD 变体结合。
