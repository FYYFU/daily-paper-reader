## 摘要
为了提升强化学习在现实世界中的适用性，对抗鲁棒强化学习领域研究如何在对抗环境扰动下训练智能体。在该设定中，主角智能体在与对手的环境扰动下优化策略，形成零和马尔可夫博弈。当对抗鲁棒强化学习与基于模型的强化学习结合时，对手可以针对学习的转移模型而非训练环境。扩展这一思想，本工作引入了推理时深度强化学习智能体的事后鲁棒化。通过将学习到的模型与训练好的标称策略结合使用，我们的方法执行了一个鲁棒策略改进步骤。目标是在不额外训练神经网络的情况下提高鲁棒性。具体地，我们利用对抗滚动下的模型预测控制，通过有界不确定性集内的投影梯度下降来近似。此外，这些离线滚动在执行时考虑并减轻了分布外问题。通过在扰动的Gymnasium MuJoCo环境中评估算法，同时考虑事后推理设定的计算限制，所提方法通过展示鲁棒性的显著提升得到验证。

## Abstract
To improve the real-world applicability of reinforcement learning (RL), the field of adversarially robust RL studies how to train agents under adversarial environment perturbations. In this setting, a protagonist agent optimizes a policy under environmental perturbations from an adversary, resulting in a zero-sum Markov game. When adversarially robust RL is combined with model-based RL, the adversary can target a learned transition model instead of the training environment. Extending this idea, this work introduces post-hoc robustification of deep RL agents at inference time. By using the learned model in combination with a trained nominal policy, our approach performs a robust policy improvement step. The goal is to improve robustness without any additional training of neural networks. Specifically, we utilize model-predictive control under adversarial rollouts, which are approximated via projected gradient descent within a bounded uncertainty set. Furthermore, these offline rollouts are performed while considering and mitigating out-of-distribution issues. The proposed methodology is validated by demonstrating significant improvements in robustness when the algorithm is evaluated in perturbed Gymnasium MuJoCo environments, while considering the computational limitations of the post-hoc inference setting.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：强化学习（RL）在从仿真到现实（sim2real）或仿真到仿真（sim2sim）的部署中，常面临环境参数扰动（如质量、摩擦系数变化），导致性能严重下降。现有对抗鲁棒RL方法（如RMDP）通常在**训练阶段**引入对手扰动，这会增加训练复杂性、引入不稳定性和性能衰减，且需要额外修改训练流程。
- **动机**：是否可以在**推理阶段**（post-hoc）直接提升预训练智能体的鲁棒性，而不改变训练过程？这允许在部署时根据需要调整鲁棒性与最优性的权衡，且无需重新训练神经网络。
- **整体含义**：本文提出一种事后鲁棒化方法，在基于模型的RL（MBRL）框架下，利用预训练的**转移模型**和**名义策略**，在推理时通过模型预测控制（MPC）和对抗性滚动（adversarial rollouts）来提高策略对扰动的鲁棒性。

## 2. 方法论
### 核心思想
- 在推理时，对每个当前状态 \(s_t\)，使用MPC优化一个局部策略 \(\pi^{\text{MPC}}_s\)。该MPC基于学习到的转移模型 \(p_\theta\) 进行**悲观（pessimistic）滚动**：每个预测的下一状态通过**投影梯度下降（PGD）** 扰动，使其进入值函数 \(V^\pi_\psi\) 较低的邻域，模拟对手的恶意扰动。同时，通过**集成模型的不确定性估计**截断滚动深度，避免分布外（OOD）模型使用。

### 关键技术细节
1. **鲁棒优化目标**：
   - 使用一个递归定义的**滚动值函数** \(V^{\text{rollout}}_k(s_k)\)，其中下一状态的采样来自对手选择的转移：  
     \[
     s_{k+1} \sim \arg\min_{P\in\mathcal{P}} \mathbb{E}_{s'\sim P(\cdot|s,a)}[V^\pi_\psi(s')]
     \]
   - 对手的扰动通过PGD在有界L2球内实现（对应**最优输运代价（OTC）不确定性集**）。
2. **扰动生成（PGD）**：
   - 扰动 \(\delta\) 初始化为0，通过多步PGD更新：  
     \[
     \delta \leftarrow \text{Proj}_{\|\delta\|_2 \leq r_2}\left(\delta - \alpha \frac{\nabla_{(s'+\delta)} V^\pi_\psi(s'+\delta)}{\|\nabla_{(s'+\delta)} V^\pi_\psi(s'+\delta)\|_2}\right)
     \]
   - 投影确保扰动半径不超过 \(r_2\)。
3. **OOD缓解**：
   - 使用**集成模型**中成员模型预测的方差 \(\text{epistemic}(s,a)\) 来衡量不确定性。当方差超过阈值 \(h\) 时，截断滚动，避免模型外推误差。
4. **MPC策略优化**：
   - 采用**MPPI**（Model-Predictive Path Integral）算法，以学习到的策略 \(\pi_\psi\) 初始化，优化动作序列以最大化滚动值函数的期望。

### 算法流程（文字说明）
- 对于当前状态 \(s_t\)：
  - 初始化动作分布（基于 \(\pi_\psi(s_t)\)）。
  - 执行多次MPPI迭代，每次：
    - 对候选动作序列，用 \(p_\theta\) 滚动N步，但每步的下一状态均通过PGD扰动，同时用集成方差检查是否应截断。
    - 用扰动后的状态计算滚动值（最终状态用 \(V^\pi_\psi\)）。
  - 从优化后的动作分布中采样实际执行的动作 \(a_t\)。

1. 本文提出的**鲁棒MPC方法**在单参数和双参数扰动下，均显著优于原始MBPO基线，大幅减少性能衰减。
2. **对抗性扰动（PGD）** 是鲁棒性提升的关键；非鲁棒MPC有时也能带来一定改进（如Reacher），但在Hopper上效果有限，而鲁棒MPC效果更明显。

4. 滚动深度 \(N\) 和不确定性半径 \(r_2\) 存在稳定工作区域：增加N或r在一定范围内可提升鲁棒性，但过大则导致性能崩溃（模型误差累积或OOD）。
5. 方法在推理时计算开销可接受（现代GPU上每步0.1秒左右），适用于对延迟要求不极端高的场景。

## 7. 优点
- **无需重新训练**：完全在推理时进行鲁棒性提升，兼容任何预训练的MBRL智能体，不改变训练流程，避免了对抗训练的不稳定性和性能损失。
- **理论框架清晰**：基于鲁棒MDP（RMDP）和最优输运代价（OTC）不确定性集，赋予扰动有界性意义。
