## 摘要
在线策略蒸馏通过来自教师的密集、词元级监督提高了标准强化学习的学习效率。在OPD的标准KL目标中，词元级损失被均匀平均，意味着所有词元权重相等。然而，我们发现并非所有词元都是同等重要的：随着学生生成的序列变长，它们与教师分布的偏差增大，导致后续位置的监督质量下降。因此，仅使用前30%词元的OPD可以达到与使用全部词元相当的性能，而仅使用后30%词元的OPD几乎无法学习。本文通过受约束优化的视角提供了对这一问题的原理性理解。基于这些见解，我们推导出重要性加权在线策略蒸馏，其中每个词元的权重取决于学生与教师分布之间的累积差异，自然地对较早词元赋予更高权重，而对偏差较大的较晚词元降低权重。我们表明，IW-OPD收敛速度显著快于OPD，具有更高的学习效率，并且在同规模与跨规模设置下均达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9分。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

### 1. 核心问题与整体含义（研究动机和背景）

- **背景**：在线策略蒸馏（On-Policy Distillation, OPD）通过从教师模型向学生模型在线生成的序列提供密集的 token 级监督（KL 散度），大幅提升了标准强化学习（RLVR）的学习效率。标准 OPD 对所有 token 的损失进行了均匀平均。
- **核心问题**：研究发现并非所有 token 对学习同等重要。随着学生模型自生成序列长度增加，其轨迹与教师分布发生漂移，导致后期 token 上的教师监督质量严重下降，形成“位置偏差”（Position Bias）。例如，仅使用前 30% token 的 OPD 与使用全部 token 的效果相当，而仅使用后 30% token 的训练几乎无法学习。
- **研究动机**：从理论上解释该偏差，并提出一种更高效的加权方法，以重新分配有限梯度预算，主要关注早期高质量 token。

### 2. 论文提出的方法论

- **核心思想**：将 OPD 的训练过程建模为受约束的局部投影优化（trust-region）：在有限的 KL 更新预算 ρ 下，求解最优学生分布 q*，使其 KL 到教师最小化，同时保持在当前学生 πθ 的邻域内。
- **关键技术细节**：
  - 基于拉格朗日法推导出最优策略的闭式解：  
    `q*_θ(y) ∝ π_θ(y) * (π_T(y)/π_θ(y))^α`，其中 α∈(0,1) 由更新预算 ρ 决定。该解表明最优策略本质上按照学生-教师似然比对原始学生分布进行重新加权。
  - 将目标函数重写为重要性加权形式：  
    `J*_IW(θ) = - E_{y~πθ} [ Σ_t sg[˜r_t] * log(π_θ(y_t|...)/π_T(y_t|...)) ]`，其中 `˜r_t` 是归一化的前缀似然比，由 GPU 停梯度（stop-gradient）处理，作为 token 级 KL 项的乘法权重。
  - 由于原始似然比极不稳定，提出稳定化技巧：
    - 取对数并累积绝对值：`˜r_abs_t = - Σ_{k<t} |A^OPD_k|`，消除正负抵消。
    - 批次内标准化为 [0,1]：`˜r_norm_t = (d_t - d_min) / (d_max - d_min) = 1 - (Σ_{k<t}|A_k|)/(Σ_{k<T}|A_k|)`。
    - 与标准 OPD 进行插值：`˜r_IW-OPD_t = 1 + γ * (1 - Σ_{k<t}|A_k|/Σ_{k<T}|A_k|)`，默认 γ=0.5，保留标准 OPD 作为密集监督下限，附加额外预算分配给早期兼容 token。
  - 最终 token 级优势：`A^IW-OPD_t = sg[˜r_IW-OPD_t] * A^OPD_t`，使用 PPO 算法更新（算法 1）。

### 6. 论文的主要结论与发现

1. **位置偏差的存在性**：OPD 的大部分学习收益来自早期 token，后期 token 监督质量因分布漂移严重下降。仅前 30% token 即可与全量 OPD 相当。
2. **IW-OPD 提升效率与性能**：在全部测试设置中，IW-OPD 比标准 OPD 收敛更快（第 10 步即匹配或超过 OPD 最终性能），最终平均准确率提升 1~6.9 分（AIME25 最高 +6.9）。
3. **随教师增强增益更大**：更强教师（30B/235B）带来更大学生-教师分布差异，IW-OPD 通过重新分配预算有效缓解了这种低效，使更强教师变得更高效。
4. **随学生缩小增益更大**：学生越小（0.6B vs 4B），相对增益越大（相对提升 >9%），说明 IW-OPD 在跨规模蒸馏中尤其有效。
5. **与现有 OPD 变体正交**：将 IW-OPD 与 ExOPD 结合（IW-ExOPD）进一步提升了性能，表明加权思想可推广到其他奖励设计方法。
