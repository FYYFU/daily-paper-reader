## 摘要
策略内蒸馏通过密集的令牌级监督传递推理能力，但可传递信号的性质仍不清楚。我们发现推理链包含两种需要不同发现机制的知识：决策（在哪里分支），通过学生的不确定性浮现；以及证据（证明决策的中间步骤），隐藏在学生自信但错误的位置。当前方法仅捕获决策；证据令牌中的实质性知识仍未传递。我们提出DEAR（决策-证据感知推理蒸馏），首先通过学生熵识别决策，然后通过隐藏状态与决策锚点的余弦相似性发现其支持证据，并通过师生差异提升以优先处理最大的知识差距。在数学和代码基准上的三种师生配置中，DEAR始终优于标准OPD，在竞赛数学上最高提升+2.5个百分点，代码生成上提升+5.7个百分点。

## Abstract
On-policy distillation transfers reasoning ability through dense token-level supervision, yet the nature of the transferable signal remains unclear. We discover that reasoning chains contain two types of knowledge that require different discovery mechanisms: decisions (where to branch), which surface through student uncertainty, and evidence (intermediate steps that justify decisions), which hides in positions where the student is confident yet wrong. Current methods capture only decisions; the substantive knowledge in evidence tokens remains untransferred. We propose DEAR(Decision-Evidence Aware Reasoning Distillation), which first identifies decisions via student entropy, then discovers their supporting evidence through hidden-state cosine similarity to decision anchors, boosted by teacher-student divergence to prioritize the largest knowledge gaps. Across three student-teacher configurations on math and code benchmarks, DEAR consistently outperforms standard OPD, with up to +2.5pp on competition math and +5.7pp on code generation.

# 论文总结：Finding the Evidence: Discovering Decision-Supporting Tokens for On-Policy Reasoning Distillation

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：在线策略蒸馏（OPD）通过密集的 token 级监督（学生生成轨迹，教师提供对数概率）将推理能力从教师转移到学生。然而，现有方法对所有 token 平等训练或仅选择高熵 token（如 Wang et al., 2026），虽然捕捉了“决策”token（如逻辑连接词、分支标记），却忽略了支撑决策的实质性 “证据” token（如中间推导步骤、API 调用），导致学生模型在正确决策后仍执行错误（如错误替换、错误计算）。
- **核心问题**：推理链中存在两种知识，需要不同的发现机制。决策 token 可通过学生不确定性（熵）显露；证据 token 则隐藏在学生自信但错误的位置，熵对其不可见。因此，现有蒸馏方法仅传递了“骨架”，遗漏了“血肉”，推理能力不完整。
- **研究意义**：揭示了蒸馏信号的结构化分解，提出了一种能够同时捕获两种知识的方法，从而提升了推理蒸馏的效果，尤其在困难数学和代码生成任务上。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：提出 DEAR（决策-证据感知推理蒸馏），分为两阶段：
  1. **决策标识**：基于学生熵选取 top-p% 高熵 token 作为决策集 D。
  2. **证据发现**：对非决策 token，计算其与学生深层隐藏状态与决策锚点的余弦相似度，并结合师生 logprob 差异（知识差距）作为证据显著性分数，选取 top-q% 作为证据集 E。最终训练损失只作用于 S = D ∪ E。
- **关键技术细节**：
  - 学生熵 $H^S_t = -\sum_v \pi_\theta(v|x,y_{<t}) \log \pi_\theta(v|x,y_{<t})$。
  - 师生差异 $\delta_t = |\log \pi_\theta(y_t) - \log \pi_T(y_t)|$。
  - 证据关联度 $a_j = \max_{i\in D} \cos(h^L_i, h^L_j)$，其中 $h^L$ 是学生最深层的隐藏状态。
  - 归一化后，证据显著性分数 $s_j = \hat{a}_j \times (1+\hat{\delta}_j)$。
  - 使用 PPO 风格的策略梯度损失，仅对 S 计算。
- **算法流程**（见 Algorithm 1）：学生 rollout → 计算熵、差异、隐藏状态 → 决策集 D → 对非决策 token 计算证据分数 → 选取证据集 E → 返回 S = D ∪ E → 更新策略。

## 6. 主要结论与发现

1. **推理链包含两类知识**：决策（熵高）和证据（熵低但差异大）。证据 token 在梯度质量中占重要比重（DEAR 覆盖 75.8% vs 决策仅 39.1%）。
2. **DEAR 显著优于 Standard OPD**：在数学竞赛（AIME 2024/2025）上提升最高（+2.2x 至 +2.7x），代码生成 +5.74pp on MBPP。
3. **知识差距越大收益越大**：跨族配置（Setting B）的增益最大，因为教师知识与学生内部化差异大。
4. **证据发现需结合语义关联与知识差距**：纯余弦相似度或纯差异选择均不如二者组合。
5. **选择比例具有稳健性**：p=0.2 和 q=0.2 是最优，退化是渐进的。

## 7. 优点

- **创新性**：首次将推理链知识分解为决策与证据，并针对性设计两阶段选择机制，解决了熵基方法的盲点。
- **有效性**：在多个困难基准上取得一致提升，尤其AIME这类需要长链推理的任务，证明证据转移是必要的。
- **分析深入**：通过Lorenz曲线、token向量可视化、分类恢复统计等方法，实证验证了证据的存在和DEAR的效果。
- **轻量高效**：仅需学生一次前向传播的隐藏状态，证据发现为简单矩阵乘法，计算开销可忽略。
- **泛化性**：在数学和代码两个领域均有效，且跨不同师生配置稳定。

## 8. 不足与局限

- **计算资源需求**：需存储学生最后一层隐藏状态（每样本一层），增加显存占用；虽然论文认为可控，但对长序列大批量仍可能成为瓶颈。
- **领域覆盖有限**：仅验证数学和代码，未涉及其他推理类型（如常识、科学、多模态）。决策-证据结构在其他领域是否普遍存在尚不明确。
- **固定选择比例**：p和q在训练中固定，未探索自适应调度（如随训练推进增加证据覆盖），可能限制最终性能。
