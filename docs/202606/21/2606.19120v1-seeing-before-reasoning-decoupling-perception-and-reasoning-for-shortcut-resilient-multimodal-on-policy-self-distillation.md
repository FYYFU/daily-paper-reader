## 摘要
在线策略自蒸馏（OPSD）在模型自身的轨迹上训练，并使用冻结副本提供基于参考目标的密集令牌级目标。这对于LLM推理效果良好，但直接扩展到多模态大语言模型（MLLMs）可能产生捷径：特权目标可能主要基于文本参考目标而非图像引导令牌。我们提出ViGOS，一种用于MLLM后训练的视觉引导OPSD框架。学生模型先撰写视觉描述，然后推理出最终答案。对于有效轨迹，仅图像感知教师监督描述，而特权推理教师在同一学生前缀上监督推理和最终答案。仅对无效轨迹使用参考教师以恢复输出格式。在通用视觉-语言、专家推理、视觉数学、空间定位和视觉语言先验基准测试中，ViGOS保留了OPSD的主要优势，并在易出现捷径的设置中改善了基于图像的行为。

## Abstract
On-policy self-distillation (OPSD) trains a model on its own rollouts and uses a frozen copy to provide dense token-level targets conditioned on a reference target. This works well for LLM reasoning, but a direct extension to multimodal large language models (MLLMs) can create a shortcut: the privileged target may guide tokens mainly based on the text reference target rather than the image. We propose ViGOS, a visually grounded OPSD framework for MLLM post-training. The student first writes a visual description and then reasons toward the final answer. For valid rollouts, an image-only perception teacher supervises the description, while a privileged reasoning teacher supervises the reasoning and final answer on the same student prefix. A reference teacher is used only for invalid rollouts to recover the output format. Across general vision-language, expert reasoning, visual math, spatial grounding, and visual-language-prior benchmarks, ViGOS keeps the main benefits of OPSD and improves image-grounded behavior in shortcut-prone settings.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：在线策略自蒸馏（On-policy Self-Distillation, OPSD）在纯文本大语言模型（LLMs）推理中效果显著，其核心是利用冻结的教师模型（参照目标）为学生在自己轨迹上的每个令牌提供密集的监督信号。然而，当直接将该方法扩展到多模态大语言模型（MLLMs）时，会产生一个关键的**捷径风险**：教师模型同时看到图像和参考答案，而文本信号（尤其是参考答案）往往比视觉信息更容易被模型利用，导致教师主要根据文本参考目标而非图像内容来引导学生的令牌生成。这种现象类似于视觉问答中的语言先验和捷径学习问题。

- **研究动机**：高质多模态监督数据稀缺且昂贵，OPSD通过自蒸馏利用稀疏的参考目标提供密集反馈，理论上很有吸引力。但如果直接应用，模型可能学到“答案兼容”的推理过程而忽视真实的视觉证据。因此，需要一种既能保留OPSD优势（密集、在线、自蒸馏），又能防止答案信号过早支配视觉感知的方法。

## 2. 方法论：核心思想、关键技术细节

### 核心思想：解耦感知与推理
- **结构化输出**：学生模型生成的结构化轨迹分为三个阶段：视觉描述（d）、推理过程（r）和最终答案（a），即 `y = (d, r, a)`。用 `<description>...</description>`、`<think>...</think>` 和 `\boxed{...}` 分隔。
- **分段教师监督**：对于有效轨迹（格式正确），不同段落由不同教师监督：
  - **图像仅感知教师**（`q_img`）：仅使用图像（无问题文本、无参考答案）监督描述段落 `T_d`。确保描述完全基于视觉内容。
  - **特权推理教师**（`q_rea`）：使用图像、问题和参考答案监督推理和答案段落 `T_r ∪ T_a`，但必须基于学生已经生成的描述前缀。这样答案引导只在视觉证据明确后才介入。
- **无效轨迹回退**：对于格式无效的轨迹，使用**参考教师**（`q_ref`，使用完整上下文）以反向KL散度提供恢复信号，避免格式漂移。

### 关键技术细节
- **教师模型**：所有教师均使用冻结的学生副本（自蒸馏设置），仅外部上下文不同。
- **损失函数**：
  - 感知损失：`L_perc = E[ (1-m_inv) * Σ_{t∈T_d} D_KL(q_img,t || p_θ,t) ]`
  - 推理损失：`L_rea = E[ (1-m_inv) * Σ_{t∈T_r∪T_a} D_KL(q_rea,t || p_θ,t) ]`
  - 回退损失：`L_ref = E[ m_inv * Σ_{t∈T_y} D_KL(p_θ,t || q_ref,t) ]`
  - 总目标：`L_ViGOS = λ_perc L_perc + λ_rea L_rea + λ_ref L_ref`
- **特权答案泄露率（PALR）**：提出诊断指标PALR，通过替换答案为错误答案或替换图像为不匹配图像，衡量每个令牌的修正信号中答案驱动部分占比。实验显示ViGOS的PALR显著低于标准OPSD（例如在7B模型上，推理-答案段的PALR从26.01%降至7.56%）。

## 7. 优点

- **创新性**：明确识别并解决了MLLM中OPSD的捷径风险，提出了解耦感知与推理的框架，思路简洁有效。

## 8. 不足与局限

- **描述质量风险**：生成的视觉描述可能不完整或不正确，而图像仅感知教师本身也可能产生通用不够准确的描述，错误描述会误导后续推理。
- **计算开销**：训练时需要额外的教师前向传递（三个教师），比标准OPSD更昂贵。
- **模型通用性**：仅在Qwen2.5-VL一个系列上验证，未证明在其他MLLM（如LLaVA、InternVL、Gemini系列）上的有效性。
