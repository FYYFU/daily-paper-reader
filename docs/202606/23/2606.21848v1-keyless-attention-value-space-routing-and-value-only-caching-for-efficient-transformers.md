## 摘要
我们提出了无键注意力，一种完全消除键投影的注意力机制，仅对查询和值进行操作。这产生了仅值缓存，相比标准注意力，KV缓存内存和访问开销精确减少50%，同时匹配或超越标准注意力的解码吞吐量。除效率外，我们引入了深度$m$注意力分解：标准注意力计算注意力双线性形式的深度-2分解，而无键注意力实现了该家族的深度$m$实例。当$m=3$时，无键注意力通过一个值空间路由矩阵（取代键投影）匹配标准注意力的投影矩阵数量，并引入了路由与检索之间的耦合。在五个模型和四种架构（GPT-2 280M、GPT-2 557M、Pythia 410M、Qwen2 1.5B和Llama 3.2 1B）上的实验表明，无键注意力在5个模型中的4个上困惑度匹配或优于标准QKV注意力。在下游零样本评估（GPT-2 557M）中，无键注意力在5个常识推理基准中的4个上表现更优，同时整体实现了50%的KV缓存减少。

## Abstract
We propose Keyless Attention, an attention mechanism that eliminates the key projection entirely, operating over queries and values only. This yields a Value-Only Cache that reduces KV cache memory and access overhead by exactly 50% over standard attention, while matching or exceeding standard attention's decode throughput. Beyond efficiency, we introduce Depth-$m$ Attention Factorization: standard attention computes a depth-2 factorization of the attention bilinear form, while Keyless Attention realizes a depth-$m$ instance of this family. At m=3, Keyless Attention matches the projection matrix count of standard attention via a value-space routing matrix that replaces the key projection and introduces a coupling between routing and retrieval. Experiments across five models and four architectures (GPT-2 280M, GPT-2 557M, Pythia 410M, Qwen2 1.5B, and Llama 3.2 1B) show that Keyless Attention matches or outperforms standard QKV attention on perplexity in 4 out of 5 models. On downstream zero-shot evaluation (GPT-2 557M), Keyless Attention outperforms on 4 out of 5 commonsense reasoning benchmarks, while achieving 50% KV cache reduction throughout.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：标准Transformer自注意力机制中的KV缓存（Key-Value Cache）在长序列推理中成为主要内存瓶颈。现有方法（如量化、剪枝、系统级优化）都是事后对缓存进行压缩或管理，而非质疑Key的存在的必要性。本文质疑标准QKV（查询-键-值）公式中键投影的冗余性，探索是否可以直接消除键投影，仅使用查询和值来计算注意力，从而从根源上减半KV缓存。
- **研究动机**：人类认知在检索记忆时，仅使用一个查询与存储的表示进行比对，而不需要同时存储“路由副本”和“检索副本”。标准注意力存储了每个历史token的键（用于路由）和值（用于检索），暗示了设计上的固有冗余。作者旨在重新设计注意力机制，在保持模型表达能力的同时，从源头减少KV缓存需求。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：**Keyless Attention（无键注意力）**：完全消除键投影，注意力分数直接由查询与值的点积计算，不构造键向量。公式变为：`Attention(Q, V) = softmax(Q V^T / sqrt(d_k)) V`。这带来**Value-Only Cache**：自回归推理时只缓存值向量，KV缓存内存和访问开销精确减少50%，无需近似、量化或剪枝。
- **关键技术细节**：
  1. **深度$m$注意力分解（Depth-$m$ Attention Factorization）**：将注意力双线性形式$X \Omega X^T$中的$\Omega$分解为$m$个矩阵的乘积。标准注意力（QKV）是深度$m=2$（$\Omega = W_Q W_K^T$），而Keyless Attention将$\Omega$分解为查询和值的乘积（$\Omega = \tilde{W}_Q W_V^T$），提出$m=3$的QVV(3)版本：查询由两个矩阵$W_{Q1} W_{Q2}$投影，值由$W_V$投影，投影矩阵总数与标准注意力相同（三个矩阵），但推理时可融合$W_{Q1}W_{Q2}$为一个矩阵，消除键投影计算。
  2. **值空间路由（Value-Space Routing）**：注意力分数基于查询与值的直接相似度，使路由与检索在同一空间耦合，形成更具语义意义的归纳偏置。
  3. **梯度纠缠（Gradient Entanglement）**：在QVV(3)中，$\nabla W_{Q2}$依赖于$W_V$，$\nabla W_V$依赖于$W_{Q2}$，两者相互耦合，而标准注意力中$W_K$和$W_V$的梯度完全解耦。作者假设这种耦合作为隐式正则化，减少过拟合。
- **理论等价性**：证明了单头（完全秩$W_V$）无条件等价，多头（$W_V$满列秩且满足列空间包含条件）也存在等价参数化。但Keyless Attention独立于该等价性有效。
