## 摘要
策略蒸馏（On-Policy Distillation, OPD）通过教师提供的密集、令牌级别的监督，提高了标准强化学习的学习效率。在OPD的标准KL目标中，令牌级别的损失被均匀平均，这意味着所有令牌的权重相等。然而，我们发现并非所有令牌都是平等的：随着学生 rollout 变长，它们会进一步偏离教师的分布，导致后期位置的监督质量下降。因此，仅使用前30%令牌的OPD表现可与使用全部令牌相当，而仅使用后30%令牌的OPD几乎无法学到任何东西。在这项工作中，我们通过约束优化的视角，从原理上理解了这一问题。基于这些见解，我们推导出重要性加权策略蒸馏（Importance-Weighted On-Policy Distillation, IW-OPD），其中分配给每个令牌的权重取决于学生与教师分布之间的累积差异，自然而然地提升早期令牌的权重，降低后期偏差较大的令牌权重。我们表明，IW-OPD的收敛速度明显快于OPD，具有更好的学习效率，并且在同规模和跨规模设置下均能达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9分。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

## 论文详细总结

### 1. 核心问题与整体含义（研究动机和背景）

- **位置偏差的发现**：标准在线策略蒸馏（OPD）对所有 token 的 KL 损失进行均匀平均，但实验表明，仅使用前 30% 的 token 即可达到与使用全部 token 相当的性能，而仅使用后 30% 的 token 几乎无法学习。这表明 OPD 存在严重的位置偏差（position bias）：早期 token 的监督质量高，后期 token 因学生轨迹偏离教师分布而监督质量显著下降。
- **研究动机**：OPD 中后部 token 的低质量监督浪费了有限的更新预算，导致学习效率低下。作者从约束优化的视角出发，将 OPD 视为在有限 KL 预算内将学生策略局部投影到教师策略的问题，从而解释了位置偏差的本质，并据此设计更高效的目标函数。

### 2. 方法论：核心思想、关键技术细节

- **核心思想**：将 OPD 训练建模为受信任区域约束的优化问题：
  - 目标：最小化学生与教师的 KL 散度 `D_KL(q || π_T)`，但限制学生更新后的分布 `q` 离当前策略 `π_θ` 的 KL 距离不超过预算 `ρ`。
  - 该约束问题的闭式解为最优策略 `q*_θ(y) = π_θ(y) · (π_T(y)/π_θ(y))^α / Z`，其中 `α ∈ (0,1)` 由 `ρ` 决定。解的形式表明，最优分布会对前缀似然比 `π_T(y_{<t}) / π_θ(y_{<t})` 高的 token 赋予更大权重。
- **关键技术细节与稳定化技巧**：
  - **重要性加权 OPD（IW-OPD）**：直接优化 `D_KL(q*_θ || π_T)`，通过重参数化得到 token 级别的加权目标：每个 token 的 KL 项乘以归一化的前缀重要性权重 `˜r_t`。
  - **稳定化步骤**：
    1. **对数缩放**：将似然比转换到对数空间，使用 `log(r_t) = Σ_{k<t} A_OPD_k`（`A_OPD_k` 为 token 级 OPD 优势），避免指数爆炸。
    2. **取绝对值**：为避免正负抵消，使用 `|A_OPD_k|` 积累，确保权重随序列单调递减。
    3. **标准化**：对累积绝对差异进行最小-最大标准化到 `[0,1]`，使权重在样本间可比。
    4. **与 OPD 插值**：最终权重形式为 `˜r_t = 1 + γ · (1 - (Σ_{k<t}|A_OPD_k|)/(Σ_{k<T}|A_OPD_k|))`，其中默认 `γ=0.5`，保留标准 OPD 的密集监督信号，同时将额外预算分配给兼容前缀。
- **算法流程（文字描述）**：
  1. 从当前学生策略 `π_θ` 采样 response。
  2. 计算每个 token 的 OPD 优势 `A_OPD_t`（教师 logits 与学生 logits 之差）。
  3. 使用累积的 `|A_OPD|` 计算标准化后的重要性权重 `˜r_t`。
  4. 将 `˜r_t` 乘以 `A_OPD_t` 得到 IW-OPD 优势，作为 PPO 梯度更新的信号。权重本身停止梯度（stop-gradient），仅作为乘性系数。

- **收敛后的非均匀性**：即使训练收敛，权重 `˜r_t` 仍近似 `1 - t/T` 而非均匀，存在残余偏差，可能影响最终性能的上限。
