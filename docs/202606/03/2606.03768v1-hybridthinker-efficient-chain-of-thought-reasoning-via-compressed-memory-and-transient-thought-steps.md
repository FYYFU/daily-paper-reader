## 摘要
扩展的思维链（CoT）轨迹提升了LLM的推理能力，但带来了显著的计算和内存成本。现有的CoT压缩方法通过记忆token将思考步骤压缩为紧凑表示，并在推理时仅保留这些表示来缓解这一问题，但细粒度信息的丢失使得后续步骤更容易出错。为了解决这个问题，我们提出\textbf{HybridThinker}，除了保留这些表示外，还临时保留思考步骤以提供细粒度细节。然而，我们观察到，在训练期间朴素地让后续步骤能够访问思考步骤，会使模型通过直接从这些步骤检索信息来绕过记忆token，导致模型通过记忆token进行压缩和检索的能力训练不足。因此，我们引入了一种混合训练方案，其中只有部分思考步骤通过注意力直接对后续步骤可访问，而其他思考步骤被屏蔽，迫使模型使用记忆token进行压缩和检索。在4个推理基准上，HybridThinker与未压缩的基线相匹配，在相似推理时间下，平均准确率将CoT压缩的最新水平提升了5.8个百分点。消融研究证实，临时思考步骤保留和混合训练方案都对这一提升有所贡献。

## Abstract
Extended chain-of-thought (CoT) traces improve LLM reasoning but incur substantial computational and memory costs. While existing CoT compression methods mitigate this by condensing thought steps into compact representations via memory tokens and retaining only these representations at inference time, the loss of fine-grained information makes subsequent steps more error-prone. To alleviate this, we propose \textbf{HybridThinker}, where in addition to preserved these representations, thought steps are also temporarily retained to provide fine-grained details. However, we observe that naively keeping thought steps accessible to subsequent steps \emph{during training} lets the model bypass memory tokens by retrieving information directly from these steps, leaving the model's ability to compress and retrieve information through memory tokens insufficiently trained. We therefore introduce a hybrid training scheme, in which only some thought steps are directly accessible through attention to subsequent steps, while the other thought steps are masked, forcing the model to use memory tokens for compression and retrieval. Across 4 reasoning benchmarks, HybridThinker matches the uncompressed baseline, advancing the state of the art in CoT compression by 5.8 points on average accuracy with similar inference time. Ablation studies confirm that both temporary thought-step retention and the hybrid training scheme contribute to these gains.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：扩展的思维链（Chain-of-Thought, CoT）能显著提升大型语言模型的推理能力，但其自注意力计算随序列长度二次增长，KV缓存线性增长，导致推理成本过高。
- **现有方案局限**：现有CoT压缩方法（如LightThinker）通过记忆令牌（memory tokens）将思考步骤压缩为紧凑表示，随后立即丢弃原始思考步骤的KV缓存。这虽然降低了内存和计算开销，但丢失了细粒度局部信息（如数值、约束、中间结论），使得后续步骤更容易出错，且错误会级联传播。
- **研究含义**：论文旨在保留CoT压缩的高效性，同时通过临时保留思考步骤来缓解信息丢失问题，实现更高的推理准确率。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：在推理过程中，除了保留压缩后的记忆令牌外，还临时保留原始思考步骤的KV缓存，持续服务于后续少数几个步骤（w步），为推理提供细粒度细节。
- **关键技术与流程**：
  - **推理过程**（三步循环）：
    1. **生成**：基于当前KV缓存容器生成第t个思考步骤St。
    2. **压缩**：St完成后，利用记忆令牌Mt对St进行前馈，生成压缩KV缓存 $\tilde{M}_t$。
    3. **更新KV缓存**：保留所有记忆令牌的KV缓存 $\tilde{M}_t$，同时临时保留St的KV缓存 $\tilde{S}_t$，使其可被后续w-1步访问；超过w步后丢弃。
  - **训练问题**：如果在训练时直接模仿推理的注意力模式（思考步骤对后续步骤可见），模型会过度依赖可读性更高的原始文本，而忽视通过记忆令牌压缩和检索信息，导致记忆令牌训练不足。
  - **Hybrid Attention训练方案**：
    - 在训练序列中随机划分思考步骤为两个不相交的集合：
      - **Shortcut Attention**：保留直接可见路径（模拟推理时临时保留）。
      - **Bottleneck Attention**：屏蔽直接可见路径，强制信息只能通过记忆令牌传递。
    - 这确保了压缩路径和直接路径都得到充分训练，既防止捷径学习，又消除训练-推理不匹配。
  - **损失函数**：仅对思考步骤的令牌计算标准交叉熵损失，排除问题和记忆令牌。

- HybridThinker在**Qwen2.5-7B**上匹配未压缩基线Vanilla的准确率（68.78%），同时峰值token使用减少61.7%，推理时间减少20.3%。

- **消融表明**：临时保留思考步骤是关键增益来源；混合训练（Hybrid Attention）显著优于纯Shortcut Attention或纯Bottleneck Attention训练。

## 7. 优点

- **创新性**：首次在CoT压缩中**临时保留**原始思考步骤，有效弥补信息丢失。
- **解决关键训练问题**：发现并解决了训练时的捷径学习（shortcut learning）问题，提出的Hybrid Attention训练方案巧妙平衡了压缩和直接路径。
- **全面实验验证**：在多个模型、多个数据集上进行系统比较，消融、超参数分析、案例研究充分，结论可靠。
- **实用效率**：在接近未压缩基准准确率的同时大幅降低内存和延迟，实际部署价值高。

## 8. 不足与局限

- **峰值内存增加**：相比立即丢弃思考步骤的方法，临时保留导致约10%的峰值KV缓存增加（但可通过减少记忆令牌数缓解，例如L=3时已优于LightThinker）。
- **依赖显式步骤分隔符**：方法假设每个思考步骤有明确的结束标记（如“\n\n”）；对于自由形式、无清晰边界的推理不适用，作者表示留作未来工作。
- **消融范围**：未探索是否可以将临时保留策略扩展到更长的推理链或不同模型规模（如更大或更小模型）。
- **应用限制**：目前仅针对CoT推理任务，尚未验证在其他需要上下文压缩的场景（如长文档问答）的有效性。
