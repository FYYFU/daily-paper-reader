## 摘要
针对大语言模型的强化学习（RL）通常使用标量结果奖励（例如二元正确性）来监督推理。这类奖励提供了优化方向，但很少解释模型应如何修正其错误的推理，这可能鼓励捷径学习和脆弱的策略。我们提出\textbf{SocraticPO}（苏格拉底式策略优化），这是一种策略优化框架，用苏格拉底式的自然语言指导增强强化学习轨迹。在轨迹生成过程中，学生首先独立作答；如果答案错误，教师诊断尝试并提供简洁的纠正性指导，之后学生在扩展后的上下文中继续。关键在于，这种指导与奖励衰减相结合：教师干预后获得的正确答案仅获得衰减后的奖励，防止策略将教师帮助视为获取奖励的自由途径。由于SocraticPO仅修改轨迹生成过程，同时保持标准期望奖励目标不变，因此可以插入现有的策略梯度后端（如Reinforce++）。此外，由于教师仅提供文本级别的指导，SocraticPO可以利用更强的黑盒教师模型，无需访问logits或分布匹配。在SciKnowEval的本科级别科学推理基准上，SocraticPO在强RL和自蒸馏基线上取得了改进。消融实验表明，有针对性的指导和奖励衰减都是必要的，其中奖励衰减减少了对辅助纠正的依赖。

## Abstract
Reinforcement learning (RL) for large language models usually supervises reasoning with scalar outcome rewards, such as binary correctness. Such rewards provide an optimization direction but rarely explain how a model should revise its mistaken reasoning, which can encourage shortcut learning and brittle policies. We propose \textbf{SocraticPO} (Socratic Policy Optimization), a policy-optimization framework that augments RL rollouts with Socratic-style natural-language guidance. During rollout, the student first answers independently; if the answer is incorrect, a teacher diagnoses the attempt and provides concise corrective guidance, after which the student continues under the expanded context. Crucially, this guidance is paired with reward decay: correct answers obtained after teacher intervention only receive decayed rewards, preventing the policy from treating teacher help as a free path to reward. Since SocraticPO only modifies the rollout process while leaving the standard expected-reward objective intact, it can be plugged into existing policy-gradient backends such as Reinforce++. Moreover, because the teacher provides only text-level guidance, SocraticPO can leverage stronger black-box teacher models without requiring access to logits or distribution matching. On undergraduate-level scientific reasoning benchmarks from SciKnowEval, SocraticPO improves over strong RL and self-distillation baselines. Ablations show that both targeted guidance and reward decay are necessary, with reward decay mitigating reliance on assisted correction.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有大语言模型（LLM）的强化学习（RL）后训练通常仅依赖标量结果奖励（如二元正确性），这种奖励告诉模型哪些轨迹应被强化，但不解释推理过程中具体哪一步出错以及如何修正。这容易导致“捷径学习”和脆弱策略——模型可能学会在验证器下得分而不真正掌握鲁棒推理。
- **整体含义**：受人类教育中苏格拉底式教学的启发（教师诊断学生错误并提供引导，而非直接给答案），论文提出 **SocraticPO（苏格拉底式策略优化）**，在 RL 轨迹生成（rollout）阶段注入自然语言指导，使得模型不仅能从自身错误中学习，还能通过教师的文本纠正获得过程级监督，从而提升推理策略的鲁棒性和独立性。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将 RL 的 rollout 过程改造为多轮交互：学生先独立作答；如果答案错误，教师根据学生回答提供简洁的纠正性自然语言指导；学生在此基础上继续作答。同时引入 **奖励衰减** 机制，防止模型将教师帮助视为免费奖励。
- **关键技术细节**：
    1. **交互式 Rollout**：对于每个问题，学生生成回答；若错误且未达到最大交互轮数 K，教师生成文本反馈加入对话上下文；学生继续生成新回答。交互直到回答正确或达到 K 轮。
    2. **策略优化**：保持期望奖励目标不变，仅在交互式 rollout 分布下计算。使用 **逐步骤奖励（step-wise reward）** 为每个学生尝试分配奖励，而非整个轨迹一个奖励，以精确定位哪次尝试获得正确。
    3. **奖励衰减**：对于在教师干预后（第 k≥2 步）才正确的回答，其奖励被缩放为历史平均正确奖励 μ_{k-1}；错误回答得 0 分。衰减后的奖励使得早期独立正确获得更高奖励，晚期辅助正确奖励较低，从而抑制依赖教师帮助的学习。
    4. **归一化优势性质**：论文证明了在奖励衰减下，归一化优势满足非负性、单调性（越早正确优势越大）和饱和衰减（若某一轮几乎所有样本都被纠正，优势趋于 0），这防止了“辅助奖励黑客”。

### 6. 论文的主要结论与发现

- SocraticPO 在本科级科学推理上较 RL 和自蒸馏基线取得最佳平均性能，且使用更强教师能进一步增益。
- **奖励衰减是核心**：去除奖励衰减后性能大幅下降（尤其是强教师场景），说明它有效防止模型依赖教师辅助。
- **教师指导不可或缺**：仅重试（固定提示）效果差，必须有针对性反馈。
- 教师访问真实答案不一定有益：可能导致答案泄漏或引导方式不友好，但若教师能正确使用则可能提升（如 Chemistry 上的例子）。
- SocraticPO 不导致学生向教师分布收敛（KL 散度不持续下降），说明它不是蒸馏，而是通过指导纠正学习。

### 7. 优点

- **方法创新**：将教育中的苏格拉底式引导引入 LLM 策略优化，自然语言纠正提供了过程级监督，弥补标量奖励的不足。
- **奖励衰减设计巧妙**：通过简单的历史平均缩放实现差分奖励，数学性质良好（非负、单调、饱和衰减），有效抑制辅助依赖。
- **即插即用**：仅修改 rollout 过程，保留标准期望奖励目标，可与现有策略梯度后端（如 Reinforce++）兼容。
- **支持黑盒教师**：只需教师输出文本，无需 logits 或分布匹配，可调用更强 API 模型。
- **实验分析深入**：除了主实验，还有训练动态、消融、教师信息、KL 散度等多角度分析，验证了设计合理性。
