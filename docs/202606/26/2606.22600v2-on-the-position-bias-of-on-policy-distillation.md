## 摘要
同策略蒸馏通过教师提供的密集、词元级监督，提升了标准强化学习的学习效率。在OPD的标准KL目标中，词元级损失被均匀平均，意味着所有词元权重相等。然而，我们发现并非所有词元都是平等的：随着学生轨迹变长，它们与教师分布的偏离增大，导致后期位置的监督质量下降。因此，仅使用前30%词元的OPD效果与使用全部词元相当，而仅使用后30%词元的OPD几乎学不到任何东西。本文通过约束优化的视角，对这一问题的原理提供了深入理解。基于这些洞察，我们推导出重要性加权同策略蒸馏（IW-OPD），其中每个词元的权重取决于学生与教师分布之间的累积差异，自然地对早期词元赋予更高权重，对偏离较大的后期词元降低权重。我们表明，IW-OPD的收敛速度显著快于OPD，学习效率更高，并且在同等规模和跨规模设置下均能达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9个百分点。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

# 论文《On the Position Bias of On-Policy Distillation》中文总结

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：标准同策略蒸馏（On-Policy Distillation, OPD）在训练学生模型时，对生成序列中的所有 token 施加**等权重的 KL 散度监督**。然而，学生模型自己生成的 token 序列越长，其分布越偏离教师模型，导致**后续 token 的监督质量严重下降**。
- **关键发现**：
  - 仅使用前 30% token 进行 OPD 训练，效果与使用全部 token 相当；
  - 仅使用后 30% token 训练，几乎学不到任何东西（图 1a）。
  - 教师模型在学生生成前缀上的条件正确率随着前缀变长迅速下降（图 1b）。
- **本质**：OPD 存在**位置偏差（position bias）**——早期 token 提供的监督价值远高于后期 token。作者将其归结为有限更新预算下的分配问题：每个训练步只能移动学生分布一个局部距离，因此应把有限梯度预算更多地分配给教师监督仍然可靠的前缀部分。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 核心思想：约束优化视角 → 重要性加权

- 将 OPD 实际训练视为一个**带局部信任域约束的投影问题**：
  \[
  \min_{q} D_{\text{KL}}(q \| \pi_T) \quad \text{s.t.} \quad D_{\text{KL}}(q \| \pi_\theta) \le \rho
  \]
  其中 \(\pi_\theta\) 为学生当前分布，\(\pi_T\) 为教师，\(\rho\) 为局部更新预算。
- **Proposition 1**：该问题的最优解 \(q^*_\theta\) 是一个重加权分布：
  \[
  q^*_\theta(y) \propto \pi_\theta(y) \cdot [\frac{\pi_T(y)}{\pi_\theta(y)}]^\alpha
  \]
  即按**从学生到教师的似然比**对原学生分布进行重加权，早期 token 的比值大，后期 token 比值小。

### 2.2 关键技术细节：IW-OPD 目标与稳定权重估计

- 直接优化 \(q^*_\theta\) 与教师的 KL 散度，通过**重要性采样**转化为：
  \[
  J^*_{\text{IW}}(\theta) = -\mathbb{E}_{y \sim \pi_\theta} \left[ \sum_{t=1}^T \operatorname{sg}[\tilde{r}_t] \log \frac{\pi_\theta(y_t | y_{<t})}{\pi_T(y_t | y_{<t})} \right]
  \]
  其中 \(\tilde{r}_t = \frac{r_t^\alpha}{Z_{\alpha,t}}\) 为归一化后的前缀似然比权重。
- **实际稳定的权重形式（公式 17）**：
  \[
  \tilde{r}^{\text{IW-OPD}}_t = 1 + \gamma \left(1 - \frac{\sum_{k<t} |A^{\text{OPD}}_k|}{\sum_{k<T} |A^{\text{OPD}}_k|}\right)
  \]
  其中 \(A^{\text{OPD}}_k = -(\log \pi_\theta(y_k) - \log \pi_T(y_k))\)，\(\gamma=0.5\) 为默认插值系数。该权重在 [0,1] 范围内单调递减，且使用**绝对值累积**避免符号抵消。
- **算法流程**（Algorithm 1）：
  1. 初始化学生策略 \(\pi_0\)；
  2. 每轮迭代从 \(\pi_0\) 采样一条完整响应；
  3. 计算每个 token 的教师-学生 log 概率差，得到 \(A^{\text{OPD}}_t\)；
  4. 按公式 (17) 计算前缀权重 \(\tilde{r}^{\text{IW-OPD}}_t\)，然后得到 \(A^{\text{IW-OPD}}_t = \tilde{r} \cdot A^{\text{OPD}}_t\)；
  5. 使用 PPO 更新学生策略（带重要性权重的高低版本）；
  6. 定期刷新采样策略。
