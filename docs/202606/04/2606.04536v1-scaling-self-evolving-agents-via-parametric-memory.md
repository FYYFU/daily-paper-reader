## 摘要
现有的记忆增强型LLM智能体仅在提示空间中存储过去的经验，作为文本摘要或检索到的段落，而在整个展开过程中保持模型参数冻结。这种智能体可以“查找”它们看到的内容，但无法“从中学习”：它们的策略不会因经验而改变，任何从上下文中丢弃的信息都会永久丢失。我们引入了TMEM，一个自演化的参数化记忆框架，其中智能体不仅将历史压缩成显式记忆，还通过轻量级在线更新将提炼后的监督信息吸收到快速LoRA权重Δ_t中，从而在单个片段内真正改变其未来行为。我们将其形式化为具有快速权重展开动态的智能体决策过程：动作从π_{θ_0+Δ_t}中采样，而提取动作产生监督信号来更新Δ_t以用于后续决策。这一观点使得提取策略可以通过强化学习直接优化：训练θ_0不仅改善了任务动作，还提高了用于在线LoRA适应的数据质量。我们进一步提出了基于SVD的LoRA子空间初始化以加速在线收敛。在LoCoMo、LongMemEval-S、多目标搜索和CL-Bench上的实验表明，TMEM在不同模型规模下始终优于基于摘要和基于检索的基线方法。

## Abstract
Existing memory-augmented LLM agents store past experience exclusively in prompt space, as textual summaries or retrieved passages, while keeping model parameters frozen throughout a rollout. Such agents can \emph{look up} what they have seen but cannot \emph{learn from} it: their policy is unchanged by experience, and any information dropped from the context is permanently lost. We introduce \texttt{TMEM}, a self-evolving parametric memory framework in which the agent not only compresses history into explicit memory but also absorbs distilled supervision into fast LoRA weights $Δ_t$ via lightweight online updates, genuinely altering its future behavior within a single episode. We formalize this as an agentic decision process with fast-weight rollout dynamics: actions are sampled from $π_{θ_0+Δ_t}$, while extraction actions produce supervision that updates $Δ_t$ for subsequent decisions. This view makes the extraction policy directly optimizable by RL: training $θ_0$ improves not only task actions but also the quality of the data used for online LoRA adaptation. We further propose SVD-based initialization of the LoRA subspace to accelerate online convergence. Experiments on LoCoMo, LongMemEval-S, multi-objective search, and CL-Bench show that \texttt{TMEM} consistently outperforms summary-based and retrieval-based baselines across different model scales.

现有记忆增强型LLM智能体只能将过去的经验存储为提示空间中的文本摘要或检索到的段落，而**模型参数在整个展开过程中保持冻结**。这意味着：

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想：TMEM框架
- 在展开过程中，智能体维护**工作上下文 h_t**、可选的**显式记忆 m_t** 和由LoRA权重表示的**快速参数化记忆 Δ_t**。
- 当上下文预算达到阈值时，智能体进入记忆写入模式：将当前会话提炼为基于QA的监督信号，并通过轻量级在线SFT更新应用到Δ_t。
- 后续动作从**自适应策略 π_{θ_0+Δ_t}** 中采样，其中基参数θ_0在该回合内固定，而Δ_t作为快速权重演化。

### 关键技术细节

#### 快速权重展开动态
- 定义状态s_t = (q, h_t, m_t, Δ_t)，动作a_t从策略π_{θ_0+Δ_t}中采样。

- 更新规则：m_{t_i+1} = ∅, Δ_{t_i+1} = T(Δ_{t_i}, a_{t_i}), h_{t_i+1} = ∅。T是轻量级在线SFT/LoRA更新（SGD，学习率5×10⁻⁴，5个epoch，batch size 16）。

#### SVD初始化的LoRA
- 标准LoRA随机初始化Δ₀=BA（B=0, A随机），在极少梯度步下浪费早期更新。
- TMEM对每个目标权重W进行SVD分解W=UΣVᵀ，固定投影矩阵A₀=Σ_r V_rᵀ（top-r右奇异向量乘以奇异值），只更新系数矩阵B（B₀=0）。

#### 策略优化
- 将提取动作视为决策过程的一部分，使用**停止梯度的结果奖励目标**：
  - 总目标J(θ₀) = E_τ[R(τ)]。
  - 策略梯度估计中，对Δ_t施加停止梯度（通过T的路径不反向传播），只优化θ₀。
  - 梯度同时优化普通任务动作和提取动作，使基模型学会生成对自身未来快速权重更新有用的监督数据。

## 6. 主要结论与发现

1. **TMEM在所有基准上一致优于摘要式和检索式基线**：
   - 前RL：LongMemEval-S上TMEM较最佳基线提升F1 4.79-10.21点、EM 2.34-7.02点；搜索任务提升F1约1.0-1.9点。
   - CL-Bench：TMEM总准确率32.87%（4B）和33.91%（8B），优于最佳基线的30.45%和32.18%。
   - 检索式（A-MEM）在部分任务上比摘要式（MemAgent）更差，可能因为检索质量瓶颈。
2. **RL后收益更大**：TMEM的绝对增益（F1: +4.78~+5.12；EM: +4.84~+5.02）超过其他方法，说明优化提取行为在快速权重机制下更有价值。
3. **SVD初始化显著优于随机初始化**：特别是精确匹配（EM）提升超过3点，符合理论预测。
4. **QA对形式的监督优于文本预测和自由式摘要**：尤其对需精确召回的长对话任务。
5. **冻结SVD子空间几乎无损失**：仅更新B比更新A+B性能降低很小（<0.3 F1），但参数更少、更稳定。
6. **触发预算存在最优值**：过小/过大会导致压缩丢失细节或提取困难。

## 7. 优点

- **理论贡献**：形式化了“快速权重展开”决策过程，将纯显式记忆智能体作为特例（Δ_t≡0），统一框架。

- **停止梯度RL优化**：使基模型通过RL同时优化任务行为和提取数据质量，避免通过在线更新反向传播的计算不可行问题。
