## 摘要
在策略自蒸馏中，语言模型基于特权上下文来监督自身的生成，这是稀疏奖励强化学习中一种有前景的密集监督来源。实际上，它可以实例化为一种辅助的全词汇学生到教师反向KL散度损失。因此，我们提出了SDPG，一种自蒸馏策略梯度框架，该框架结合了群体相对验证器优势、归一化标准差、精确的全词汇在策略自蒸馏以及参考策略KL正则化。实验表明，SDPG在稳定性和性能上优于RLVR和自蒸馏基线。代码可在https://github.com/lauyikfung/SDPG获取。

## Abstract
On-policy self-distillation, where a language model conditions on privileged context to supervise its own generations, is a promising source of dense supervision for sparse-reward reinforcement learning. Actually, it can be instantiated as an auxiliary full-vocabulary student-to-teacher reverse Kullback-Leibler divergence loss. We therefore propose SDPG, a self-distilled policy-gradient framework that combines group-relative verifier advantages with normalized standard deviation, exact full-vocabulary on-policy self-distillation, as well as reference-policy KL regularization. Empirically, SDPG improves stability and performance over RLVR and self-distillation baselines. The code is available at https://github.com/lauyikfung/SDPG.

# 自蒸馏策略梯度（Self-Distilled Policy Gradient）——论文详细总结

## 1. 核心问题与研究动机

- **背景**：大语言模型在后训练阶段广泛使用带可验证奖励的强化学习（RLVR），例如 GRPO。但 RLVR 存在**稀疏序列级奖励**问题——只有最终答案正确/错误信号，无法细粒度地指导每个 token 的更新；同时早期训练中大量负优势会导致不稳定。
- **现有自蒸馏方法**：在线策略自蒸馏（OPD）通过额外特权上下文（如标准答案、推理过程）为每个 token 提供密集监督。但传统蒸馏依赖更大更强的外部教师模型，增加内存开销且异构教师信号可能不稳定。
- **核心问题**：如何有效结合 RLVR 的稀疏验证奖励与在线自蒸馏的密集 token 级信号，在保持稳定性的同时提升推理性能？

## 2. 方法论：SDPG 框架

### 核心思想
- 保持一个**共享模型**：在无特权上下文时作为学生策略 \(p_t\)，在附加特权上下文时作为教师策略 \(q_t\)。
- **精确全词汇反向 KL 散度**作为自蒸馏损失 \(D_{\text{KL}}(p_t \| \text{SG}[q_t])\)，提供密集 token 级梯度。
- 与 **GRPO 式验证奖励损失**结合，并使用**参考策略 KL 正则化**（UFKL 或 URKL）保持稳定性。

### 关键技术细节
- **总体损失**：
  \[
  \mathcal{L}_{\text{SDPG}} = \mathcal{L}_{\text{out}} + \beta(k) \mathcal{L}_{\text{OPD}}^+ + \alpha \mathcal{L}_{\text{K}}(\pi_\theta, \pi_{\text{ref}})
  \]
  - \(\mathcal{L}_{\text{out}}\)：REINFORCE 风格的组相对优势损失，避免 PPO 裁剪。
  - \(\mathcal{L}_{\text{OPD}}^+\)：正优势门控的全词汇蒸馏损失（仅对 \(A_{\text{out}}>0\) 的轨迹计算）。
  - \(\mathcal{L}_{\text{K}}\)：未归一化 KL（UFKL 或 URKL），梯度无偏且稳定。

- **稳定机制**：
  - **正优势门控**：只对验证器认可（\(A>0\)）的轨迹施加蒸馏，避免在错误轨迹上强化局部合理 token。
  - **β 调度**：蒸馏系数先 warmup 后 decay，早期防止噪声，后期释放学生探索能力。

- **理论亮点**：命题 3.1 证明反向 KL 蒸馏的梯度等价于一个策略梯度更新，其中 token 优势为**中心化的 log 教师/学生比值**，这解释了自蒸馏的梯度作用机制。

## 6. 主要结论

1. **SDPG 在 AIME24/25 和 AMC23 上均取得最高准确率**（表 1），且训练曲线更早达到高奖励平台。
2. **与 RLSD 相比**，SDPG 避免了熵崩塌（RLSD 在 250 步后熵趋近 0），蒸馏信号更温和有效。
3. **消融实验证明**：
   - 蒸馏项（β）主要贡献早期快速收敛；
   - KL 正则（α）防止策略偏离过远，保持合理的响应长度和熵。
4. **小模型（1.7B）上同样验证**了有效性，且纯自蒸馏基线 OPCD 在第 250 步后出现不稳定性（准确率骤降、响应长度崩塌），表明 SDPG 的验证奖励锚定不可或缺。
