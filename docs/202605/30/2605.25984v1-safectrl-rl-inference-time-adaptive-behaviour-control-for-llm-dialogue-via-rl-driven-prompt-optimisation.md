## 摘要
确保大型语言模型（LLM）在现实部署中的安全和上下文适当行为仍然是一个关键挑战。我们提出了\textbf{SafeCtrl-RL}，一个推理时的行为控制框架，能够在无需模型重新训练或参数修改的情况下实现自适应安全调节。该方法将对话生成建模为序列决策过程，其中强化学习代理根据上下文反馈动态选择提示调整策略。通过迭代优化抑制不安全行为，我们将其概念化为推理时的行为遗忘。在多个LLM和不安全对话场景下的评估中，SafeCtrl-RL持续提高了安全性和响应质量，优于现有的基于提示的优化方法，并实现了良好的性能-效率权衡。**警告：本文可能包含有害语言的示例，建议读者自行判断。**

## Abstract
Ensuring safe and contextually appropriate behaviour in Large Language Models (LLMs) remains a critical challenge for real-world deployment. We present \textbf{SafeCtrl-RL}, an inference-time behavioural control framework that enables adaptive safety regulation without model retraining or parameter modification. The method formulates dialogue generation as a sequential decision process, where a reinforcement learning agent dynamically selects prompt adjustment strategies based on contextual feedback. This allows unsafe behaviours to be suppressed through iterative refinement, which we conceptualise as inference-time behavioural unlearning. Evaluated across multiple LLMs and unsafe dialogue scenarios, SafeCtrl-RL consistently improves safety and response quality, outperforms existing prompt-based optimisation methods, and achieves favourable performance--efficiency trade-offs. **Warning: This paper may contain examples of harmful language, and reader discretion is recommended.

## 2. 论文提出的方法论
### 核心思想
- 将对话生成建模为**马尔可夫决策过程（MDP）**，一个强化学习智能体（DQN）在每个推理步骤中根据当前状态（36维向量）选择一个动作（11种提示调整策略之一），生成修改后的系统提示，然后由目标LLM生成回复，经安全-质量评估器打分，产生奖励信号，指导下一轮迭代。循环直到满足停止条件（评分≥0.9）或达到最大迭代次数。

### 关键技术细节
- **状态空间**：36维向量，包含（1）元学习特征（精炼进度、探索率），（2）得分特征（近期均值、方差、策略历史效果），（3）提示特征（预测危害类别、风险等级、结构特征、易失性哈希），综合反映对话动态和优化进程。
- **动作空间**：11种离散的提示调整策略，分为三类：
  - 直接或零历史访问：minimal, raw_history
  - 基于摘要的策略：AI_summary_only, AI_enhanced, Progressive_summary, Hybrid
  - 性能与轨迹感知策略：Best-worst-recent, Performance_tiered, Trajectory_focused, Contrast_learning, Adaptive_performance
- **奖励函数**：
  - 基本奖励：`r(q,s) = q^{αβ} * s^{(1-α)β}`，其中α=0.6（安全-质量权衡），β=10（尖锐度）
  - 最终奖励：若关键安全指标（如毒性、仇恨）低于阈值θ=0.8，则奖励置零；否则使用基本奖励。这样强制安全约束不可被质量替代。
- **安全-质量评估器**：基于DeepEval框架，使用Gemini 2.0 Flash进行评分，包含质量维度（连贯性、答案相关性等）和安全维度（12类危害，基于Ji et al. 2025的细粒度分类）。
