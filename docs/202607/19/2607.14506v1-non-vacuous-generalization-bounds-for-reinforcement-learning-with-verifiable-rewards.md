## 摘要
尽管基于可验证奖励的强化学习（RLVR）被广泛用于提升大型语言模型（LLMs）的推理能力，但所得模型的泛化性仍鲜为人知。在这项工作中，我们首次为十亿参数规模下的参数高效RLVR微调建立了非平凡的泛化界。我们的方法将PAC-Bayes压缩界适应于此设置，并通过应用Gumbel-max重参数化技巧来处理Token生成固有的随机性。为了使这些界可操作化，我们提出了渐进式RLVR框架，它结合了RLVR与在线蒸馏、TinyLoRA和模型量化。渐进式RLVR在经验上保留了标准LoRA微调的84-97%的性能，同时产生的模型可压缩性提高了14,796倍。我们展示了该框架在四个领域（数学问题求解、编程、常识推理和Text-to-SQL）中产生了非平凡的泛化界。我们的界比基础模型的准确率高出9-51%，并且与微调模型的准确率相差在6-11%以内。

## Abstract
While reinforcement learning with verifiable rewards (RLVR) is widely used to improve the reasoning capabilities of large language models (LLMs), the generalizability of the resulting models remains poorly understood. In this work, we establish the first non-vacuous generalization bounds for parameter-efficient RLVR fine-tuning at the billion-parameter scale. Our approach adapts PAC-Bayes compression bounds to this setting, and addresses the inherent stochasticity of token generation by applying the Gumbel-max reparameterization trick. To operationalize these bounds, we propose the Progressive RLVR framework, which integrates RLVR with on-policy distillation, TinyLoRA, and model quantization. Progressive RLVR empirically retains 84-97% performance of standard LoRA fine-tuning while producing models that are 14,796x more compressible. We show that this framework yields non-vacuous generalization bounds in four domains: mathematical problem-solving, programming, general-knowledge reasoning, and Text-to-SQL. Our bounds exceed the accuracy of the base model by 9-51% and lie within 6-11% of the accuracy of the fine-tuned models.

- **核心问题**：基于可验证奖励的强化学习（RLVR）广泛用于提升大语言模型（LLM）的推理能力（如数学、编程），但微调后模型的**泛化性缺乏理论理解**。现有工作表明 RLVR 模型可能过度拟合训练样本，甚至出现多样性崩溃等病态行为，因此需要严格的泛化界来验证其可靠性。
- **研究背景**：经典复杂度度量（如 VC 维、Rademacher 复杂度）在十亿参数模型上给出平凡（vacuous）界；PAC-Bayes 压缩界在预训练上取得了非真空界，但扩展至 RLVR 面临两大挑战：
    1. RLVR 的奖励定义在随机生成的 Token 序列上，输出空间是序列长度 × 词表大小的组合大空间，导致经验奖励不可计算。
    2. 除训练数据分布外，模型自身的生成过程引入了第二重随机性，使现有 PAC-Bayes 界无法直接适用。
- **整体含义**：本文首次为十亿参数规模的 LLM 经参数高效 RLVR 微调后**建立了非真空的泛化界**，为 RLVR 模型在下游部署（尤其是高风险场景）提供了形式化的可靠性验证。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式/算法流程

- **核心思想**：
    - 将 RLVR 的泛化误差分解为**外部误差**（prompt 分布上的期望 vs 经验）和**内部误差**（模型生成随机性带来的蒙特卡洛估计误差），分别采用 Hoeffding 不等式和 PAC-Bayes 压缩界进行约束。
    - 利用 **Gumbel-max 重参数化**将 Token 生成表示为输入、随机噪声和假设的确定性函数，从而将生成随机性外部化为假设独立的噪声分布，使得压缩论点可用。
    - 为满足界所需的极端压缩，提出 **Progressive RLVR 流水线**：先训练高容量教师模型（标准 LoRA），再通过**在线蒸馏（on-policy distillation）**将教师能力压缩至使用 **TinyLoRA**（仅含少量可训练标量）的学生模型，最后量化 TinyLoRA 适配器以进一步降低描述长度。

- **关键技术细节**：
    - **PAC-Bayes 压缩界**：采用 Solomonoff 先验（先验概率与假设的 Kolmogorov 复杂度成反比），界的形式为  
      \( R(h) \ge \frac{1}{mn}\sum_{i,j} r(x_i, g(x_i,\xi_{i,j};h)) - \Delta \left( \sqrt{\frac{1}{n}} + 1 \right) \sqrt{\frac{C(h)\log 2 + 2\log C(h) + \log(2/\delta)}{2m}} \)，其中 \(C(h)\) 是假说的压缩比特数。
    - **误差分解**：外部误差 ~ \(O(1/\sqrt{m})\)（m 为训练 prompts 数），内部误差 ~ \(O(1/\sqrt{mn})\)，当 \(n = \Omega(m)\) 时内部误差可忽略。
    - **在线蒸馏**：教师模型在推理时生成中间 Token 级别的监督信号，使学生模型学习复合奖励的稠密信息，比直接 RLVR with TinyLoRA 的奖励高约 10%。
    - **TinyLoRA**：基于预训练权重的截断 SVD 分解和固定随机投影矩阵，将权重更新参数化为低维向量 v（最低仅 1 个标量），结合量化（5 级，2.3 bits）实现极端压缩。
    - **奖励子采样**（Corollary 1）：可从训练数据中均匀子采样 \(m_0\) 个 prompt 估计经验奖励，仅增加 \(O(1/\sqrt{m_0})\) 的附加项。

- **算法流程**（文字说明）：
    1. **教师 RLVR 训练**：在标准 RLVR 目标下训练高容量 LoRA 教师。
    2. **在线蒸馏**：教师对 prompt 生成 response 时提供 Token 级 KL 散度损失；学生模型（TinyLoRA）在在线策略（on-policy）下同时学习最大化奖励与匹配教师分布。
    3. **量化**：将训练好的 TinyLoRA 适配器参数量化为均匀离散等级，进一步减少比特数。
    4. 计算压缩界：使用 Krichevsky–Trofimov 算术编码计算实际描述长度，代入 bound。
