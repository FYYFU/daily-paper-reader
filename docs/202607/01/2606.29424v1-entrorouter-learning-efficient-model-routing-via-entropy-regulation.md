## 摘要
模型路由通过在能力不同的模型之间进行选择，平衡了解决方案的准确性和计算成本。虽然最近的多轮框架将推理与规划交织在一起，但我们识别出一种结构性的失败模式，称为信任区域坍塌。我们证明，推理与路由的深度耦合，加上稀疏监督下强预训练先验的主导作用，会导致退化局部最优，使有能力的专家被系统性压制。为了解耦这些过程，我们提出EntroRouter，一种将熵调节作为核心目标的单轮路由框架。我们首先通过软监督初始化策略，拟合一个合适模型的分布，以建立一个用于探索的高熵先验。随后，我们使用软锚点稳定强化学习，该锚点利用离线能力估计在安全的信任区域内协调受控的熵收缩。大量实验表明，EntroRouter保留了最强专家模型98.3%的准确率，同时将计算成本降低了48.25%。

## Abstract
Model routing balances solution accuracy and computational cost by selecting among models of varying capabilities. While recent multi-round frameworks interleave reasoning and planning, we identify a structural failure mode termed Trust Region Collapse. We demonstrate that the deep coupling of reasoning and routing, exacerbated by the dominance of strong pre-training priors under sparse supervision, leads to degenerate local optima where capable experts are systematically suppressed. To decouple these processes, we propose $\textbf{EntroRouter}$, a single-round routing framework that treats entropy regulation as a core objective. We first initialize the policy via Soft Supervision, fitting a distribution of suitable models to establish a high-entropy prior for exploration. Subsequently, we stabilize Reinforcement Learning using a Soft Anchor, which utilizes offline capability estimates to orchestrate controlled entropy contraction within a safe trust region. Extensive experiments demonstrate that EntroRouter retains 98.3% of the strongest expert's accuracy while reducing computational costs by 48.25%.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心挑战**：大型语言模型部署中存在精度-成本权衡，大模型推理能力强但成本高，小模型高效但处理复杂任务失败。模型路由旨在动态选择最成本效益的模型来处理每个查询。
- **现有方法的缺陷**：近期多轮路由框架（如Router-R1）将路由决策与推理执行深度耦合，导致一种结构性失败模式——**信任区域坍塌（Trust Region Collapse）**。具体表现为：小路由器无法理解大专家生成的复杂推理轨迹（因轨迹相对于路由器自己的训练分布是分布外的），在稀疏结果奖励下，强化学习优化器观察到高成本却无性能增益的专家调用，从而迫使路由器将调用外部专家的概率降至零，退化为“永不升级”的次优策略。
- **研究目标**：解耦路由与推理，通过显式管理策略分布的熵来避免过早坍塌，实现高效的单轮路由。

## 2. 方法论：核心思想、关键技术细节、算法流程

- **核心思想**：将熵调节作为一等目标，通过两阶段训练实现**受控的熵收缩**——先保持高熵以探索，再在安全信任区内锐化决策。
- **阶段 I：软监督（Soft Supervision）SFT**
  - **目标**：初始化一个高熵先验，避免强制拟合单个“最佳”模型导致的过早决策边界锐化。
  - **流程**：
    1. 用弱探针模型（Qwen3-1.7B）评估问题难度，门控值g(x)指示简单问题（g=1）或复杂问题（g=0）。
    2. 基于离线能力估计（Monte Carlo rollout，N=5）计算每个模型的pass rate κ(m,x)，定义最佳专家mbest为满足κ≥τ的最便宜模型，以及top-k最经济合格模型集合Mtopk。
    3. 构建目标分布Q(m|x)：简单问题时塌缩到mbest（硬标签）；复杂问题时均匀分配到Mtopk（软标签），保持概率质量扩散。
    4. 最小化KL散度L_SFT = -ΣQ log π_θ。
- **阶段 II：带有软锚点（Soft Anchor）的强化学习**
  - **目标**：在安全信任区内实现决策锐化，动态优化精度-成本权衡。
  - **流程**：
    1. 基于离线pass rate和成本定义**参考分布π_ref**：μ = κ – α·C，T控制锐度。
    2. 使用GRPO优化，目标函数为：max E[r] – β D_KL(π||π_ref) – β H(π)（数学上等价于奖励整形为 r + β log π_ref）。
    3. 奖励函数r(m,x)使用期望效用（dense signal）而非稀疏二元结果，并设计分段式的R_succ：对于比mbest便宜的模型（风险区）钳制为0.6；对于比mbest贵的模型（效率区）施加线性惩罚。
    4. 熵惩罚项–βH(π)鼓励锐化，但受D_KL项约束，避免过早崩溃。
- **关键公式**：软锚点机制通过拉格朗日乘子法导出，等价于将π_ref的log似然加入奖励。
