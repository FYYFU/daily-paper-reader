## 摘要
现代大语言模型（LLM）在后训练中依赖强化学习来推动特定能力，但将多种能力整合到一个模型中仍然困难。现有方法如异策略微调（Off-Policy Finetune）和混合强化学习（Mix-RL）要么效率低下，要么性能损失。在这项工作中，我们提出了多教师同策略蒸馏（MOPD），这是一种结合多个领域强化学习教师能力的后训练范式：我们首先运行每个领域的专门强化学习以获得一组领域教师，然后通过这些教师对学生自己生成的轨迹进行蒸馏。这消除了暴露偏差并提供了密集的优化信号。在Qwen3-30B-A3B上，MOPD优于混合强化学习、级联强化学习、异策略微调和参数合并（Param-Merge）基线，几乎继承了每位教师的全部能力。MOPD还支持领域教师的并行独立开发，消除了多领域后训练中典型的跨领域耦合。MOPD已部署在工业级前沿模型MiMo-V2-Flash的后训练中，展示了其在前沿规模大语言模型中能力融合的实用价值。

## Abstract
Modern large language models (LLMs) rely on reinforcement learning during post-training to push specific capabilities, yet integrating multiple capabilities into one model remains hard. Existing methods, such as Off-Policy Finetune and Mix-RL, are either inefficient or lose performance. In this work, we propose Multi-teacher On-Policy Distillation (MOPD), a post-training paradigm for combining the capabilities of multiple domain RL teachers: we first run per-domain specialised RL to obtain a set of domain teachers, then distill these teachers into the student on its own rollouts. This eliminates exposure bias and provides a dense optimization signal. On Qwen3-30B-A3B, MOPD outperforms Mix-RL, Cascade RL, Off-Policy Finetune, and Param-Merge baselines, inheriting nearly all of each teacher's capability. MOPD also enables parallel, independent development of domain teachers, removing the cross-domain coupling typical of multi-domain post-training. MOPD has been deployed in the post-training of MiMo-V2-Flash, an industrial-scale frontier model, demonstrating its practical value for capability integration in frontier-scale LLMs.

### MOPD: 面向大语言模型后训练中能力融合的多教师在线蒸馏 论文分析

#### 1. 核心问题与整体含义

-   **研究动机**：现代大语言模型（LLM）的后训练通常依赖于针对特定领域（如数学推理、代码生成、指令遵循）的强化学习（RL）来提升模型在该领域的能力。然而，实际应用中需要一个模型在所有领域都表现出色。如何高效地将多种不同领域的能力融合到单一模型中，是当前LLM后训练面临的一个尚未解决的挑战。
-   **现有方法的局限**：论文指出了四种主流融合范式的缺陷：
    -   **混合强化学习（Mix-RL）**：将不同领域的提示混合在一起训练，导致跨领域训练信号相互干扰（跷跷板效应），联合模型性能低于各领域的专家模型。
    -   **级联强化学习（Cascade RL）**：按顺序训练不同领域，早期领域的能力会在后续训练中被遗忘，且过长的训练链路增加了稳定性风险。
    -   **离线微调（Off-Policy Finetune）**：用专家模型生成的轨迹进行监督微调（SFT），会产生典型的暴露偏差问题（训练和推理时的分布不匹配）。
    -   **参数合并（Param-Merge）**：在权重空间合并专家模型，融合后的模型通常不稳定，无法完全继承所有专家的全部能力。
-   **核心问题**：如何在消除暴露偏差、提供密集优化信号的同时，实现稳定、高效且可并行的多领域能力融合。

#### 2. 方法论

-   **核心思想**：提出一种名为**多教师在线策略蒸馏（MOPD）** 的全新后训练范式。MOPD的核心是将能力融合从权重空间或静态数据集空间，转移到**策略空间（Policy Space）** 中进行。具体而言，它先将多个领域专家作为冻结的“教师”，然后训练一个“学生”模型，在其自身的在线生成轨迹上向这些教师进行蒸馏学习。

-   **技术细节**：MOPD分为三个阶段：
    1.  **通用监督微调（SFT）**：在涵盖所有目标领域的广泛语料上微调基座模型，获得一个共享的SFT检查点。这个检查点是后续所有教师的初始化，也是MOPD阶段学生的初始化。
    2.  **领域专门化RL训练**：对于每个领域（d），从共享的SFT检查点出发，独立且并行地训练一个专门的领域RL专家模型（教师 $\pi_{\phi_d}$）。每个专家可以使用最适合其领域的RL策略（如数学用可验证答案RL，软件工程用可执行沙盒RL）。
    3.  **MOPD蒸馏**：学生模型从SFT检查点初始化，冻结领域专家作为教师组。在多领域数据集上，每步优化过程如下：
        1.  从数据集中采样一批提示。
        2.  学生模型根据每个提示生成一条完整轨迹。
        3.  根据提示所属领域，将其轨迹路由到对应的领域专家教师。
        4.  教师模型在学生的轨迹上进行预填充，计算每个token的log概率。
        5.  学生模型通过最小化**逐token的反向KL散度**来更新。

-   **关键公式与算法**：
    -   **损失函数（L_rev-KL）**：优化目标是学生分布与教师分布之间的反向KL散度。
        -   `L = E[ (1/|y|) * Σ Σ π_θ(v) * log(π_θ(v) / π_φ_d(v)) ]`
    -   **策略梯度实现**：梯度可以转化为策略梯度形式。其中，教师与学生log概率的差值充当**逐token优势函数（^A_MOPD,t）**，可以直接集成到现有的PPO/GRPO框架中。
        -   `^A_MOPD,t = sg[ log π_φ_d(y_t) - log π_θ(y_t) ]`
    -   **Top-k蒸馏实现**：为减少方差和计算开销，提出只对教师模型Top-k个token进行蒸馏，并添加一个修正项（`π_φ_d(v) - π_θ(v)`）以确保优化的正确性。
    -   **基础设施**：领域教师作为独立的预填充服务，异步运行在学生采样之后。由于教师预填充与其他序列的采样可以重叠，MOPD的墙钟时间主要由采样决定，教师的计算开销基本被隐藏。

#### 7. 优点

-   **方法层面**：
    -   **消除暴露偏差**：学生在其自身的rollout分布上训练，完美匹配了推理时的状态分布。
