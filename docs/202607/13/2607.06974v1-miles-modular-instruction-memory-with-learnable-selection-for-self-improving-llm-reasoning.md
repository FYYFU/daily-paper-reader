## 摘要
大语言模型在测试时通过额外计算日益提高了推理能力，然而现有大多数工作孤立地处理每个问题。当问题顺序出现时，跨问题累积可重用的经验可以进一步提升性能。现有的基于记忆的方法要么存储完整解决方案模板，这类模板对新问题的泛化能力差，要么使用启发式的步骤级选择，其优化目标并非最终答案的正确性。学习选择策略需要大规模训练数据和固定的动作空间，使得此类方法不适用于记忆逐步扩展且仅有限监督可用的测试时设置。我们提出了MILES（具有可学习选择功能的模块化指令存储器，用于自我改进的大语言模型推理），这是一个在现实测试时约束下动态扩展逐步记忆并应用正确性优化记忆组合的框架。MILES维护由非对称子目标嵌入和子指令对组成的模块化记忆单元，每个单元关联一个可学习的选择头。这种记忆结构实现了由粗到细的检索机制：粗粒度级别允许记忆扩展，并从置信样本中收集用于训练选择头的监督信号；细粒度阶段应用学到的选择头对粗粒度候选进行重排序，并为不确定样本引导推理。MILES始终匹配或超越先前方法，同时实现了优越的准确性与效率权衡。大量实验证明了其有效性、鲁棒性和可迁移性。

## Abstract
Large language models (LLMs) increasingly improve their reasoning at test time via additional computation, yet most existing works treat each problem in isolation. When problems arrive sequentially, accumulating reusable experience across them can further improve performance. Existing memory-based methods either store whole-solution templates that generalize poorly to novel problems or use heuristic step-level selection that is not optimized for final-answer correctness. Learning selection policies requires large-scale training data and fixed action spaces, making such approaches unsuitable for test-time settings where memory expands incrementally and only limited supervision is available. We propose MILES (Modular Instruction Memory with LEarnable Selection for self-improving LLM reasoning), a framework that dynamically expands step-wise memory and applies correctness-optimized memory composition under realistic test-time constraints. MILES maintains modular memory units consisting of asymmetric pairs of sub-goal embeddings and sub-instructions, each associated with a learnable selection head. This memory structure enables a coarse-to-fine retrieval mechanism: The coarse level enables memory expansion and collects supervision for training selection heads from confident samples, while the fine stage applies learned selection heads to rerank coarse-level candidates and guide reasoning for uncertain samples. MILES consistently matches or outperforms prior methods while achieving superior accuracy-efficiency tradeoffs. Extensive experiments demonstrate its effectiveness, robustness, and transferability.

## 2. 方法论

### 核心思想
- **MILES（Modular Instruction Memory with LEarnable Selection）**：维护一个由非对称子目标嵌入和子指令对组成的模块化记忆，每个记忆单元关联一个可学习的选择头。通过“粗到细”两阶段检索机制，在测试时动态扩展记忆并应用正确性优化的记忆组合。

### 关键技术细节
1. **记忆单元结构**：每个记忆单元为 `(子目标嵌入, 子指令文本)` 不对称对。子目标嵌入作为检索键，子指令自然语言文本用于条件生成。记忆单元通过k-means聚类从置信轨迹中总结得到，聚类距离结合步骤嵌入和步骤目标嵌入的余弦相似度。
2. **粗到细检索机制**：
   - **Layer 1（粗检索）**：基于子目标相似度，利用余弦相似度从记忆中检索候选单元，并应用每个单元的有效阈值排除无关项。若无匹配，则直接生成新步骤并作为种子用于后续记忆扩展。
   - **Layer 2（细粒度学习选择）**：每个记忆单元有一个轻量级二元分类器作为选择头，输入是从推理状态中提取的上下文特征向量（通过上下文码本和最大相似度池化得到），输出适用性分数。最终分数结合Layer 1相似度和Layer 2学习分数，通过超参数平衡。
3. **选择头学习**：
   - 将记忆选择形式化为序列决策问题，目标最大化最终答案正确性。推导出在KL约束下的最优策略闭式解，但实际使用二元交叉熵损失训练每个选择头，正例为选择该单元导致正确最终答案的状态，负例为错误或不同单元正确的情况。
   - 数据收集通过MCTS在置信样本上进行，仅使用Layer 1相似度采样（避免偏倚）。每收集足够数据后训练选择头。
4. **增量测试时记忆构建与推理**：
   - **置信样本**：通过多数投票评估答案一致性，用于扩展记忆和训练选择头。利用MCTS收集训练数据，总结新记忆单元。
   - **不确定样本**：使用记忆引导的树搜索，通过两阶段检索选择子指令，逐步扩展推理链，最终多数投票决定答案。

## 6. 主要结论与发现

- MILES在全部6个基准和4个骨干上一致匹配或超越基线，在数学和学术任务上均取得增益。
- 在更小规模开源模型上提升更显著（如GPT-OSS-20B、Qwen3-30B-Instruct）。
- 精度-效率权衡中，MILES随计算预算增加持续改进，而基线很快饱和，表明跨问题记忆积累优于孤立搜索。
- 跨模型迁移有效：利用辅助模型（甚至更小模型）构建的记忆可转移至更强骨干，提升性能。
- 两阶段选择机制显著优于仅相似度或仅学习选择，且学到的选择头优于状态相似度重排序或LLM提示重排序。
- 样本顺序影响在数据量少时存在，但足够数据后几乎消失。
- 更多MCTS rollout带来更大记忆和更高准确率。
- 学到的子指令选择确实提高了中间步骤的指令正确率和最终答案正确率（表14）。

## 7. 优点

- **创新性**：将记忆选择从启发式提升为学习驱动的正确性优化，且兼容增量扩展，无需参数更新、外部监督或固定动作空间。
- **模块化设计**：非对称记忆结构（嵌入+文本）支持高效检索和灵活生成；每单元独立选择头支持新单元添加而不重训旧头。
- **自监督流程**：利用置信样本自动生成伪标签，通过MCTS收集密集步骤级监督，避免人工标注。
- **泛化性强**：跨数据集、跨模型、跨样本顺序均表现稳定，不同数学和学术任务上均有效。
- **可解释性**：记忆以文本形式存储，可被检查；案例展示子指令如何引导正确推理方向。
- **效率权衡**：通过粗到细检索，在保证性能的同时控制计算开销；KV缓存进一步降低开销（对开源模型）。

## 8. 不足与局限

- **计算开销**：依赖MCTS rollout收集训练数据，在置信样本上增加额外计算。但作者指出可通过并行处理、KV缓存、辅助模型生成轨迹来缓解。
- **依赖指令遵循能力**：假设LLM能正确理解和执行子指令，对指令遵循弱的模型可能效果有限。
- **数据需求**：虽然测试时有限数据可用，但极端少样本场景（如每类仅1-2个样本）下选择头可能过拟合。论文通过正则化、集成等策略缓解，但未在极低数据量下评估。
- **实验覆盖**：消融实验主要集中在AIME系列，未在所有数据集上复现。真实应用中的噪声（如伪标签错误）可能影响训练质量，未充分探讨。
