## 摘要
可解释性仍然是强化学习中的一个关键问题。当动作空间连续时，从在复杂环境中训练的智能体中提取可解释策略尤其具有挑战性。我们提出ORCAID，一种从运行在具有连续动作空间的混合连续离散环境中的强化学习智能体中提取可解释规则策略的新方法。我们的主要贡献是一种高效的斜决策树训练算法，该算法通过超平面划分状态空间并拟合局部线性模型。关键思想在于三阶段分裂搜索：高效的随机初始化、局部优化和后向消除。最后，合并相邻叶节点以生成描述给定深度强化学习策略的简洁可解释规则集。我们在多个强化学习环境中评估ORCAID，证明所提取的基于规则的策略以较少的参数保持了强大的性能，甚至可用于改进原始深度强化学习策略的性能。

## Abstract
Explainability remains a key issue in reinforcement learning (RL). Distilling an interpretable policy from an agent trained in a complex environment is particularly challenging when the action space is continuous. We introduce ORCAID, a novel method for extracting interpretable rule-based policies from RL agents operating in mixed continuous-discrete environments with continuous action spaces. Our main contribution is an efficient oblique decision tree training algorithm that partitions the state space by hyperplanes and fits local linear models. The key idea lies in a three-stage split search: efficient random initialization, local refinement, and backward elimination. Finally, adjacent leaves are merged to yield a concise set of interpretable rules describing a given deep RL policy. We evaluate ORCAID across multiple RL environments, demonstrating that the extracted rule-based policies maintain strong performance with a low number of parameters and can even be used to improve the performance of the original deep RL policy.

### 2. 论文提出的方法论

**核心思想**：通过斜决策树（Oblique Decision Tree）划分状态空间，在每个叶节点拟合线性回归模型，然后合并相邻叶节点并简化条件，最终生成由“规则体（条件）→ 规则头（线性模型）”构成的规则集。

**关键技术细节**：

1. **数据收集**：
   - 收集原始策略与环境交互的轨迹。
   - 使用K-means聚类（肘部法选取k）以减少高密度区域过采样。
   - 从簇中心添加高斯噪声重采样，生成更均匀的样本，并查询原始策略获取动作标签。

2. **树学习**（算法1）：
   - **斜分裂条件**：使用线性不等式 `w·x ≤ b` 作为分裂条件，可同时处理连续特征和二元特征（通过权重约束）。
   - **三阶段分裂搜索**：
     1. **随机初始化**：在PCA降维后的空间中随机选取pca_dim个样本点，求解线性系统 `Aw = -1` 生成候选超平面。
     2. **局部优化**：若候选的加权MSE（基于子节点OLS回归误差）仍高于阈值 `max_mse`，则使用差分进化（Differential Evolution）优化权重。
     3. **后向消除**：在叶节点中，对每个动作维度的OLS回归执行后向特征消除（移除系数绝对值小于ϵ或去除后MSE增量小于δ的特征）。
   - **终止条件**：达到最大深度、样本数少于 `min_spl`、或叶节点MSE低于 `max_mse`。

3. **简化与合并**：
   - 定义“简单区域”（叶节点）和“区域”（多个简单区域的并集）。
   - 通过“活动条件”和“相邻区域”的概念，迭代合并满足MSE阈值的相邻区域（阈值τ = 2·max_mse，相对改进容忍ϵ=0.05）。
   - 合并后的区域可表示为析取范式（DNF），并运用Quine-McCluskey方法进行布尔条件简化。
   - 最后对每个区域的线性模型再次执行后向消除。
   - **模型大小度量**：`||M|| = Mconds + MBools + Mregrs`，分别对应条件系数非零个数、条件个数、回归系数非零个数。

4. **DAgger迭代改进**（算法2）：
   - 执行当前代理模型，收集其与原始策略动作偏差大的状态。
   - 聚类+重采样后加入训练集，重新训练树（从上一深度扩展一层）。
   - 持续迭代直至性能不再提升或无可收集数据。

### 5. 实验数量与充分性

- **数量**：对每个环境，每种方法均测试多组超参数（如树深度、规则数），并选择最佳模型进行比较。主实验覆盖9个环境，每个环境9次运行，统计均值和标准误。消融实验（附录G）在Pendulum和Swimmer上分析了数据量、噪声、聚类数、线性回归、DAgger、简化等组件的影响。策略改进实验对每个环境采用四种镜像策略，每种随机选取50个变换，进行统计显著性检验。LLM评测使用了6个模型、15个问题。
- **充分性**：实验设计较为全面，对比了多种主流方法，控制了采样和DAgger的公平性，并进行了消融和额外应用分析。但消融仅在两个环境上完成，未在全部环境进行，略有限制。整体上实验客观、公平，结果支撑了主要结论。

### 6. 论文的主要结论与发现

1. **性能与复杂度**（RQ1）：ORCAID以远小于决策树和Cubist的模型大小实现了同等或更优的回报比率，RuleFit在部分环境性能不足。
2. **保真度**（RQ2）：ORCAID的测试MSE通常最低，说明对原策略近似最好。
3. **策略改进**（RQ3）：基于ORCAID规则的镜像策略能更频繁地发现原始RL策略的弱点，并实现更高的相对改进幅度（例如在Half Cheetah上Best=5.5%）。
4. **可解释性**（RQ4）：LLM-as-a-Judge评分显示，ORCAID在大多数环境（15个问题平均分4.0左右）显著优于决策树和Cubist（平均分2-3），尤其在模型全局结构、物理意义、调试便利性方面获得更高评价。
5. **消融**：线性回归、DAgger、区域简化、聚类与重采样等组件均对最终性能有贡献；去除这些组件会导致性能下降或模型增大。

### 7. 优点

- **方法创新**：首次将斜决策树与线性回归结合用于连续动作RL策略提取，三阶段搜索高效且能处理高维状态。
- **模型简洁**：通过合并区域和布尔简化，生成的规则集参数数量远少于轴对齐树，且保持可解释的物理意义。
- **实用价值**：不仅提取可解释策略，还能用于提升原始RL策略，展示了行动能力（actionability）。
- **评估全面**：除了常规指标，还采用了LLM-as-a-Judge进行可解释性评估，减少了主观偏差，并提供了交互式可视化工具。
- **开源实现**：提供代码仓库，便于复现和扩展。

### 8. 不足与局限

- **实验覆盖**：消融研究仅在两个环境上执行，未在全9个环境验证各组件贡献。策略改进实验未在Pendulum和Mountain Car（已最优）上进行，且部分环境（如Reacher）未找到改进。
- **可扩展性**：当前实验最大状态空间17维（Half Cheetah），动作空间6维；更高维问题可能需要额外的降维或特征选择。
- **策略类型限制**：方法针对确定性策略设计，对随机策略仅能通过独立高斯模型扩展，未在当前论文中验证。
- **LLM评估可信度**：虽然使用6个LLM，但与人类评判的一致性尚未验证，可能受提示设计和模型偏见影响。
- **模型大小度量**：参数数量虽能反映部分复杂度，但未考虑条件语义的易理解性（如布尔简化后的可读性），LLM评分可作为补充。
- **计算成本**：ORCAID训练时间（尤其是复杂环境如Hopper 30分钟）比决策树长，但比Cubist（22分钟）稍高，且在可接受范围内。
